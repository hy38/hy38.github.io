### Exercise : Calculus (Basic)

###### 1) Plot the function $y = f(x) = x^3 - \dfrac{1}{x}$ and its tangent line when $x = 1$.

Lets first define the function y in python. After, what I have to do is finding the tangent line of y when x is 1.

The derivative functinon y' is $3x^2 + \dfrac{1}{x^2}$, and the tangent line would be $y = ax + b$. We can get `a`, `b` as below.

1. y'(1) equals to 3 + 1 = 4, which is `a`. So the tangent line is $y = 4x + b$.

2. Using the fact that y, y' passes cordinate (1,0), `b` comes to be `-4`.

So the tangent line is $y = 4x - 4$.

Using python, we can simply plot it.

```python
def f2(x):
    return x ** 3 - 1 / x

x2 = np.arange(0, 3, 0.1)
plot(x2, [f2(x), 4 * x2 - 4], 'x2', 'f2(x)', legend=['f2(x)', 'Tangent line (x2=1)'])
```

###### 2) Find the gradient of the function $f(x) = 3x_1^2 + 5e^{x_2}$.

By differentiating the function with respect to each variable $x_1$, $x_2$ can we get the gradient of f(x).

$\frac{\partial f}{\partial x_{1}} = 6 x_{1}$

$\frac{\partial f}{\partial x_{2}} = 5 e^{x_{2}}$

$\nabla f(x) = (6 x_{1}, 5 e^{x_{2}})$

###### 3) What is the gradient of the function $f(\mathbf{x}) = \|\|\mathbf{x}\|\|_2$?

Since $ \|\|\mathbf{x}\|\|_2 = (x_1^2 + x_2^2 + ...)^\frac{1}{2} $,

when you partially derivate each $x_i$s,

you can get the answer : $\nabla_{\mathbf{x}} \|\|\mathbf{x}\|\|_2 = \frac{\mathbf{x}}{\|\|\mathbf{x}\|\|_2}$

###### 4) Can you write out the chain rule for the case where $u = f(x, y, z)$ and $x = x(a, b), y = y(a, b)$, and $z = z(a, b)$?

$\frac{\partial u}{\partial a}=\frac{\partial u}{\partial x}\frac{\partial x}{\partial a}+\frac{\partial u}{\partial y}\frac{\partial y}{\partial a}+\frac{\partial u}{\partial z}\frac{\partial z}{\partial a}$

$\frac{\partial u}{\partial b}=\frac{\partial u}{\partial x}\frac{\partial x}{\partial b}+\frac{\partial u}{\partial y}\frac{\partial y}{\partial b}+\frac{\partial u}{\partial z}\frac{\partial z}{\partial b}$
