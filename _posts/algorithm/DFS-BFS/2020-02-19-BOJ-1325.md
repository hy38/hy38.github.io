> DFS를 이용한 문제해결

# [문제링크](https://www.acmicpc.net/problem/1325)

- 이 문제는 DFS를 활용하여 지나갈 수 있는 모든 정점들의 최대경우를 찾는 문제이다.
- 완전탐색으로 쉽게 풀리는 쉬운 문제이다.
- 단방향 그래프라는 것에 유의하고, 화살표 방향이 u가 v를 신뢰하면 v->u 모양이 됨에 유의하자.

# 알고리즘 세우기

1) 입력받으면서 단방향 그래프를 완성한다.

1-1) 이 때, 입력이 0이 아닌 1부터 들어오므로, 여유있게 vector를 N+1로 resize해준다.

2) 1번 정점부터 N번 정점까지 DFS를 반복하면서, 몇개의 노드에 접근하는지 cnt변수를 이용하여 센다.

2-1) cnt변수를 mem배열에 저장한 후, cnt변수의 최댓값을 구한다.

3) 문제가 원하는 출력대로, 최대값을 갖는 정점들의 인덱스를 오름차순으로 출력한다.

## 1. 입력받으면서 단방향 그래프를 완성한다.
  
```c++
void input()
{
    cin.sync_with_stdio(false);
    cin.tie(NULL);

    cin >> N >> edges;
    adj.resize(N + 1);
    vis.resize(N + 1);

    for (int i = 0; i < edges; ++i) {
        cin >> u >> v;
        adj[v].push_back(u);    //  단방향 그래프임에 유의!
    }
}
```

- 인접리스트를 입력받아 단방향 그래프를 완성한다.

### 1-1. 이 때, 입력이 0이 아닌 1부터 들어오므로, vector를 N+1로 resize해준다.

- 

```c++
    adj.resize(N + 1);
    vis.resize(N + 1);
```

## 2. 1번 정점부터 N번 정점까지 DFS를 반복하면서, 몇개의 노드에 접근하는지 cnt변수를 이용하여 센다.

- DFS를 이용해서 몇 개의 정점을 방문할 수 있는지를 카운팅한다.

```c++
void DFS(int node)
{
    vis[node] = true;

    for (int i = 0; i < adj[node].size(); ++i) {
        int next = adj[node][i];

        if (!vis[next]) {
            cnt++;
            DFS(next);
        }
    }
}
```

- DFS의 리턴값을 이용한 카운팅도 가능해보인다.

```c++
    return 1 + DFS(next);
```

- 자신에게 편하고, 가독성 좋고, 짜임새 있는 코드를 사용하자.

### 2-1. cnt변수를 mem배열에 저장한 후, cnt변수의 최댓값을 구한다.

```c++
    for (int i = 1; i <= N; ++i) {
        vis = vector<bool>(N + 1, false);   //  방문배열 초기화.
        cnt = 0;
        DFS(i);

        mem[i] = cnt;
        max_cnt = max(max_cnt, cnt);
    }
```

## 3. 문제가 원하는 출력대로, 최대값을 갖는 정점들의 인덱스를 오름차순으로 출력한다.

- 그냥 속시원하게 cnt의 모든값들을 배열에 저장하였다.
- 최댓값이 갱신될때마다 vector에 추가하는(기존의 것은 clear하고) 방법도 존재할것같다.

```c++
    for (int i = 1; i <= N; ++i) {
        if (mem[i] == max_cnt)
            cout << i << " ";
    }
```

# 전체 소스코드

```c++
#include <algorithm>
#include <iostream>
#include <vector>
using namespace std;

int N, edges, u, v;
int mem[10004], cnt, max_cnt;
vector<vector<int>> adj;
vector<bool> vis;

void input()
{
    cin.sync_with_stdio(false);
    cin.tie(NULL);

    cin >> N >> edges;
    adj.resize(N + 1);
    vis.resize(N + 1);

    for (int i = 0; i < edges; ++i) {
        cin >> u >> v;
        adj[v].push_back(u);
    }
}

void DFS(int node)
{
    vis[node] = true;

    for (int i = 0; i < adj[node].size(); ++i) {
        int next = adj[node][i];

        if (!vis[next]) {
            cnt++;
            DFS(next);
        }
    }
}

int main()
{
    input();
    for (int i = 1; i <= N; ++i) {
        vis = vector<bool>(N + 1, false);
        cnt = 0;
        DFS(i);

        mem[i] = cnt;
        max_cnt = max(max_cnt, cnt);
    }

    for (int i = 1; i <= N; ++i) {
        if (mem[i] == max_cnt)
            cout << i << " ";
    }

    return 0;
}
```
