> DFS와 BFS를 이용한 문제해결

# [문제링크](https://www.acmicpc.net/problem/11559){: target="_blank"}

- 이 문제는 4개 이상의 연속된 정점들을 찾아 후속작업을 해주는 문제이다.

- 연속된 정점들을 찾는 방법에는 DFS, BFS가 있다.

- 방문하지 않은 정점들에 대해 완전탐색을 통해 연속된 정점들을 찾는다.

- 그 후 큐에 해당 좌표들을 저장하여 후속작업을 처리하였다.

- 후속작업이라 함은 그 정점들을 '.'으로 지워주는 것과 그 정점들의 해당 column의 인덱스배열에 
카운팅을 해주는 것이다.

- 각 column별 몇 개의 정점이 사라지는지 카운팅해주는 이유는, 이후에 있는 '중력'을 편하게 적용하기 위해서이다.

# 알고리즘 세우기

1) 맵을 입력받는다.

2) '.'(점)이 아니거나 방문하지 않은 모든 정점들에 대해 DFS를 하여 같은 **알파벳이 4개 이상이 연속되어있는지** 판별한다.

2-1) DFS를 할 때마다 정점의 좌표를 **큐**에 넣고 카운트를 세어 4개 이상일 경우 flag를 들어준다. 

2-2) flag가 올라가 있을경우, 즉 4개 이상 연속된 알파벳이 존재한다면, 그 점들을 큐에서 빼어 **'.'으로 바꾸어준다.** 그리고 각 column별 **몇 개의 점이 사라지는지를 카운트**해준다.

2-3) flag가 올라가 있지 않을 경우에는 큐에 있는 **모든 좌표들을 뺀다.** 모든 정점들에 대해 flag가 올라가지 않을 경우에는 더이상 연속된 4개 이상의 정점이 없으므로 **반복을 중지한다.**

3) '중력'을 이용해 **떠있는 정점들을 밑으로 내린다**. 이때 각 column별 몇 개의 정점이 '.'으로 바뀌었는지를 countBlank 배열에 저장해둔 것을 이용한다.

4) 이번 반복에서 연속된 4개 이상의 정점이 하나라도 나왔다면 반복을 계속한다. 나오지 않았다면, 반복을 중지한다.

5) 정답을 출력한다.


## 1. 맵을 입력받는다.
  
```c++
void input()
{
    cin.sync_with_stdio(false);
    cin.tie(NULL);
    for (int r = 0; r < row; ++r) {
        for (int c = 0; c < col; ++c) {
            cin >> map[r][c];
        }
    }
}
```

- 이 문제의 핵심은 크게 **두 가지** 인 것 같다. 하나는 **연속된 4개 이상의 정점들을 '.'(점)으로 바꿔주는 것**, 다른 하나는 **중력을 적용하여 붕 뜬 정점들을 내려주는 것** 이다.


## 2. '.'(점)이 아니거나 방문하지 않은 모든 정점들에 대해 DFS를 한다.

- 다음은 DFS를 진행하는 코드이다. 아래에서 각 코드에 대해 설명이 있다.

```c++
bool isSafe(int r, int c)
{
    if (0 <= r && r < row && 0 <= c && c < col)
        return true;
    return false;
}

void DFS(int r, int c)
{
    if (l >= 4)
        flag = true;

    vis[r][c] = true;
    for (int i = 0; i < 4; ++i) {
        int nr = r + dr[i], nc = c + dc[i];
        if (!isSafe(nr, nc))
            continue;

        if (map[r][c] == map[nr][nc]) {
            if (!vis[nr][nc]) {
                q.push({nr, nc});
                l++;
                DFS(nr, nc);
            }
        }
    }
}
```

### 2-1. DFS를 할 때마다 정점의 좌표를 큐에 넣고 카운트를 세어 4개 이상일 경우 flag를 들어준다.

- **l**이라는 변수를 이용하여 **4개 이상 연속된 정점**이 있는지 파악한다.

```c++
void DFS(int r, int c)
{
    if (l >= 4)
        flag = true;    //  flag를 올려 다음 작업을 준비한다.

    if (map[r][c] == map[nr][nc]) {

                q.push({nr, nc});   //  큐에 일단 넣어준다.
                l++;    //  연속된 정점이 하나 더 늘었음을 표시해준다.

}

int main()
{

                if (map[r][c] != '.' && !vis[r][c]) {
                    flag = false;
                    l = 1;

                    DFS(r, c);

                    ... some code ...
}
```

### 2-2. flag가 올라가 있을경우, 즉 4개 이상 연속된 알파벳이 존재한다면, 그 점들을 큐에서 빼어 '.'으로 바꾸어준다. 

- 그리고 아래에서 중력을 용이하게 적용하기 위해 각 column별 **몇 개의 점이 사라지는지**를 카운트해준다.

- 이 때 iter변수는 **전체 반복문**을 다시 돌지 결정하는 일종의 flag이다. 이 flag는 모든 정점에 대해 DFS를 했을 때, 4개 이상의 연속된 정점이 존재할 경우 올라간다.

```c++
int main(){

    ... some code ...

    if (flag == true) {
        q.push({r, c});
        while (!q.empty()) {
            int r = q.front().first;
            int c = q.front().second;
            q.pop();
            map[r][c] = '.';
            countBlank[c]++;
            iter = true;
        }
    }

    ... some code ...

}
```

### 2-3. flag가 올라가 있지 않을 경우에는 큐에 있는 모든 좌표들을 뺀다.

- 큐에는 4개 이상 연속된 정점들의 좌표가 들어가기 때문에, **연속되지 않은 정점의 좌표가 담겨있으면 안된다**.

```c++
int main(){

    ... some code ...
    
    else {  //  연속된 4개의 정점이 존재하지 않음!
        while (!q.empty())
```

## 3. '중력'을 이용해 떠있는 정점들을 밑으로 내린다.

- 이 함수를 구현하면서 고민을 많이 했었다. **메모리의 읽기가 비효율적으로 작동**하기 때문이었다. 하지만 이 코드가 **가장 깔끔하게 맞아떨어지고**, 문제의 시간제한을 **초과할정도의 오버헤드도 아니기 때문에** 이대로 사용하였다.

- 각 column별 몇 개의 정점이 '.'으로 바뀌었는지를 countBlank 배열에 저장해둔 것을 이용한다.

- 그리고, 각 column을 도는 반복문의 마지막에 초기화해주는 코드를 넣어보았다.

```c++
void pushDownMap()
{
    for (int c = 0; c < col; ++c) {
        //  push down
        for (int i = 0; i < countBlank[c]; ++i) {
            for (int j = 10; j >= 0; --j) {
                if (map[j][c] != '.' && map[j + 1][c] == '.') {
                    map[j + 1][c] = map[j][c];
                    map[j][c] = '.';
                }
            }
        }
        countBlank[c] = 0;  //  initialize array countBlank to 0
    }
}

int main()
{
    input();
    while (1) {
        
        ... some code ...

        pushDownMap();

        ... some code ...
    }
}
```

## 4. 이번 반복에서 연속된 4개 이상의 정점이 하나라도 나왔다면 반복을 계속한다. 나오지 않았다면, 반복을 중지한다.

- 위에서 4개 이상의 정점이 하나라도 존재하는지의 flag를 통해 무한 반복문의 **탈출 여부를 결정**한다.

- 모든 정점들에 대해 flag가 올라가지 않을 경우에는 더이상 연속된 4개 이상의 정점이 없으므로 반복을 중지한다.

```c++
}
```

# 고민한 것들 / 알게된 것들

## DFS vs BFS

- 아직도 이 문제를 다시 풀라고 하면 고민이 참 많이 될 것 같다. 나에게는 DFS가 더 직관적이고 보기 편하기 좋은 그래프 탐색방법이기 때문이다. 그래서 이 문제를 풀 때 탐색은 DFS를 이용하되, queue를 같이 이용하여 해결하였다.

- 다만, BFS를 사용하게 되면 **조금 더 깔끔한 코딩이 가능**하지 않을까 싶다. 탐색과 지워야할 정점을 기록해두는것이 보다 더 유기적으로 작동할 것 같다.

- 하나 재미있는 얘기는, 맨 처음 이 문제를 BFS로 접근했다는 것이다. (ㅋㅋ)

## 꼭 큐여만 하는가?

- 문제풀때는 막연하게 큐를 써야겠다 생각을 했었다. 아마 BFS의 영향이 아닐까 싶다.

- 하지만 지금 생각해보니, 큐가 아닌 벡터, 스택, 배열 등의 **다른 자료구조로도** 좌표를 저장할 수 있을 것 같다. 다시말해, 연속된 정점들의 좌표를 저장하는데에 있어 **큐의 기능이 필요가 없는 셈이다.**

## 문제해결방법 선택의 근거

### 완전탐색

- 이 문제를 완전탐색할 수 있는 근거는 map의 크기가 **12 * 7로 작고, 고정되어있기** 때문이다. 따라서 완전탐색을 다 해봤자 **84번의 for문을** 돌기 때문에 큰 고민없이 접근했다.

# 전체 소스코드

```c++
#include <iostream>
#include <queue>
#include <string.h>
using namespace std;
char map[14][8];
bool vis[14][8], flag;
int row = 12, col = 6, ans, l;
int countBlank[7];
int dr[4] = {1, 0, -1, 0};
int dc[4] = {0, -1, 0, 1};
queue<pair<int, int>> q;

void input()
{
    cin.sync_with_stdio(false);
    cin.tie(NULL);
    for (int r = 0; r < row; ++r) {
        for (int c = 0; c < col; ++c) {
            cin >> map[r][c];
        }
    }
}

bool isSafe(int r, int c)
{
    if (0 <= r && r < row && 0 <= c && c < col)
        return true;
    return false;
}

void DFS(int r, int c)
{
    if (l >= 4)
        flag = true;

    vis[r][c] = true;
    for (int i = 0; i < 4; ++i) {
        int nr = r + dr[i], nc = c + dc[i];
        if (!isSafe(nr, nc))
            continue;

        if (map[r][c] == map[nr][nc]) {
            if (!vis[nr][nc]) {
                q.push({nr, nc});
                l++;
                DFS(nr, nc);
            }
        }
    }
}

void pushDownMap()
{
    for (int c = 0; c < col; ++c) {
        //  push down
        for (int i = 0; i < countBlank[c]; ++i) {
            for (int j = 10; j >= 0; --j) {
                if (map[j][c] != '.' && map[j + 1][c] == '.') {
                    map[j + 1][c] = map[j][c];
                    map[j][c] = '.';
                }
            }
        }
        countBlank[c] = 0;
    }
}

int main()
{
    input();
    while (1) {
        bool iter = false;
        memset(vis, false, sizeof(vis));
        for (int r = 0; r < row; ++r) {
            for (int c = 0; c < col; ++c) {
                if (map[r][c] != '.' && !vis[r][c]) {
                    flag = false;
                    l = 1;

                    DFS(r, c);
                    if (flag == true) {
                        q.push({r, c});
                        while (!q.empty()) {
                            int r = q.front().first;
                            int c = q.front().second;
                            q.pop();
                            map[r][c] = '.';
                            countBlank[c]++;
                            iter = true;
                        }
                    } else {
                        while (!q.empty())
                            q.pop();
                    }
                }
            }
        }

        pushDownMap();

        if (iter)
            ans++;
        else
            break;
    }
    cout << ans << "\n";
    return 0;
}
```
