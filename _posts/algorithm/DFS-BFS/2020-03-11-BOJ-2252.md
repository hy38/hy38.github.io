> 위상정렬을 이용한 문제해결

# [문제링크](https://www.acmicpc.net/problem/2252)

- 이 문제는 위상정렬을 이용하여 학생들의 키를 정렬하는 문제이다.

- 문제에서 제시되는 입력으로는 학생 A가 학생 B보다 큰지 여부만 주어진다.

- 따라서 각 학생의 키가 주어졌을 경우 사용할 수 있는 일반적인 정렬(Merge sort 등)의 사용이 어렵다.

- **문제 특성상 사이클이 존재하는것이 불가능하다.** 누구의 키가 더 큰지 명확하다.

- 그래프 알고리즘 중 **위상정렬**을 이용하면, 화살표로 문제에서 제시하는 입력을 그대로 표현할 수 있고 한 번의 DFS로 문제를 해결할 수 있다.

- 위상정렬에 대한 기본적인 이해는 [여기](/topological-sort){: target="_blank"}를 통해 확인할 수 있다.

- 이 문제의 시간복잡도는 O(V + E)이다.

# 알고리즘 세우기

1) 벡터들을 알맞게 할당한 후, 인접리스트로 **방향 그래프**를 입력받는다.

2) **위상정렬**을 수행한다.

2-1) DFS가 끝나는 정점들을 **스택**에 넣어준다.

3) 스택을 하나씩 **pop**하여 정답을 출력한다.

## 1. 벡터들을 알맞게 할당한 후, 인접리스트로 방향 그래프를 입력받는다.
  
```c++
void input()
{
    cin.sync_with_stdio(false);
    cin.tie(NULL);
    cin >> N >> M;

    adj.resize(N + 1);
    vis.resize(N + 1);

    int u, v;
    for (int i = 0; i < M; ++i) {
        cin >> u >> v;
        adj[u].push_back(v);    //  방향 그래프이므로 화살표 방향에 유의.
    }
}
```

- 문제에 맞게 **방향 그래프**와 **스택**을 만들어준다.

## 2. 위상정렬을 수행한다.

```c++
int main(){
    
    ... some code ...
    
    for (int i = 1; i <= N; ++i) {
        if (!vis[i])    //  if not visited
            DFS(i);
    }

    ... some code ...

    return 0;
}
```

### 2-1. DFS가 끝나는 정점들을 스택에 넣어준다.

- 지극히 일반적인 DFS 함수에 **종료시** **스택에 push하는** 코드만 추가하였다.

- DFS를 살짝만 바꿔도 위상정렬이 가능하다는게 신기하다. 물론 증명이 따로 있지만.

```c++
void DFS(int node)
{
    vis[node] = true;
    for (int i = 0; i < adj[node].size(); ++i) {
        int next = adj[node][i];
        if (!vis[next])
            DFS(next);
    }
    s.push(node);   //  DFS함수가 끝나기 직전에 스택에 정점을 넣어준다.
}
```

## 3. 스택을 하나씩 pop하여 정답을 출력한다.

- 스택을 이용한 이유는 스택의 **pop** 기능을 사용하기 위함이다.

- **DFS가 끝나는 순서의 역순**이 곧 위상정렬 순서이기에, 역순으로 만들기 용이하게 pop 기능이 있는 **스택 자료구조**를 사용하였다.

```c++
    while (!s.empty()) {
        cout << s.top() << " ";
        s.pop();
    }

    return 0;
```


# 알게된 것들

- 위상정렬을 구현하는 방법 중 queue를 이용하는 방법이 속도가 미묘하게 조금 더 빠른 것 같다.
  
  - 큐를 이용하는 방법은 indegree가 0인 정점들을 큐에 집어넣는 방법이다.

  - 이 방법 역시 [여기](/topological-sort){: target="_blank"}를 참조하면 좋을것같다.

- 큐가 더 빠르게 작동하는 이유는 **재귀함수를 사용하지 않아서** 인것같다.

# 전체 소스코드
```c++
#include <iostream>
#include <stack>
#include <vector>
using namespace std;

stack<int> s;
vector<vector<int>> adj;
vector<bool> vis;
int N, M;

void DFS(int node)
{
    vis[node] = true;
    for (int i = 0; i < adj[node].size(); ++i) {
        int next = adj[node][i];
        if (!vis[next])
            DFS(next);
    }
    s.push(node);
}

void input()
{
    cin.sync_with_stdio(false);
    cin.tie(NULL);
    cin >> N >> M;

    adj.resize(N + 1);
    vis.resize(N + 1);

    int u, v;
    for (int i = 0; i < M; ++i) {
        cin >> u >> v;
        adj[u].push_back(v);
    }
}

int main()
{
    input();
    for (int i = 1; i <= N; ++i) {
        if (!vis[i])
            DFS(i);
    }

    while (!s.empty()) {
        cout << s.top() << " ";
        s.pop();
    }

    return 0;
}
```
