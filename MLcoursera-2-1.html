Linear Regression with Multiple Variables

Intro

м•Ҳл…•н•ҳм„ёмҡ”. мқҙлІҲ нҸ¬мҠӨнҠём—җм„ңлҠ” multiple variables(features)лҘј к°Җм§„ м„ нҳ•нҡҢк·Җм—җ лҢҖн•ҙ м•Ңм•„ліҙкІ мҠөлӢҲлӢӨ. м§ҖлӮң нҸ¬мҠӨнҠём—җм„ң н•ҳлӮҳмқҳ variableмқ„ к°Җм§ҖлҠ” м„ нҳ•нҡҢк·ҖлҘј кіөл¶Җн–ҲлҠ”лҚ°, к·ё м—°мһҘм„ мқҙлқјкі  ліҙмӢңл©ҙ лҗ©лӢҲлӢӨ. мҡ°лҰ¬лҠ” м§ҖлҸ„н•ҷмҠөм—җм„ң м„ нҳ•нҡҢк·Җ, л¶„лҘҳлҘј кіөл¶Җн•ң лӢӨмқҢ л№„м§ҖлҸ„н•ҷмҠө л“ұ лҚ”мҡұ лӢӨм–‘н•ң кІғл“Өмқ„ кіөл¶Җн•  кІғмһ…лӢҲлӢӨ. л”°лқјм„ң мқҙ нҸ¬мҠӨнҠё лӢӨмқҢмңјлЎңлҠ” кё°ліём Ғмқё Octave м„ӨлӘ…мқ„ лӢӨлЈЁкі , кі§л°”лЎң л¶„лҘҳ(Classification)мқ„ кіөл¶Җн•ҳкІҢ лҗҳкІ мҠөлӢҲлӢӨ.

Notation

лЁјм Җ, мң„мҷҖ к°ҷмқҖ лҚ°мқҙн„°лҘј лҙҗмЈјм„ёмҡ”. мқҙм „мқҳ нҸ¬мҠӨнҠём—җм„ңлҠ” лӢЁмҲңнһҲ м§‘мқҳ нҒ¬кё°м—җ лҢҖн•ң к°ҖкІ©мқ„ мҳҲмғҒн•ҳлҠ” м„ нҳ•нҡҢк·ҖлҘј л°°мӣ лӢӨл©ҙ, мқҙлІҲм—җлҠ” лҚ”мҡұ кө¬мІҙм ҒмңјлЎң л“Өм–ҙк°Җм„ң лӢӨм–‘н•ң featuresлҘј кё°л°ҳмңјлЎң м§‘к°ҖкІ©мқ„ мҳҲмғҒн•ҳкІҢ лҗ©лӢҲлӢӨ. к·ём—җ м•һм„ңм„ң н‘ңкё°лІ•м—җ лҢҖн•ҙ м •лҰ¬н•ҳкІ мҠөлӢҲлӢӨ.

  • xмқҳ л°‘мІЁмһҗлҠ” к°Ғк°Ғмқҳ featureл“Өмқ„ мқҳлҜён•©лӢҲлӢӨ. мҳҲлҘјл“Өм–ҙ м№ЁмӢӨмқҳ к°ңмҲҳм—җ кҙҖн•ң лҚ°мқҙн„°лҠ” $x_2$мһ…лӢҲлӢӨ.
  • yлҠ” мқҙм „кіј к°ҷмқҙ output variable, мҰү м§‘к°ҖкІ©мқ„ мқҳлҜён•©лӢҲлӢӨ.
  • m : мқҙм „м—җ м„ӨлӘ…н•ңкІғкіј к°ҷмқҙ exampleмқҳ к°ҜмҲҳлҘј л§җн•©лӢҲлӢӨ. мң„мқҳ н…Ңмқҙлё”м—җм„ңлҠ” к°Ғк°Ғмқҳ rowл“Өмқҙ к°Ғ м§‘л“Өмқҳ лҚ°мқҙн„°мқҙкё° л•Ңл¬ём—җ, мҙқ 4к°ңмқҳ м§‘ exampleмқҙ мһҲм–ҙ mмқҖ 4мһ…лӢҲлӢӨ.
  • n : featuresмқҳ к°ҜмҲҳл“Өмһ…лӢҲлӢӨ. мң„ н…Ңмқҙлё”м—җм„ңлҠ” нҒ¬кё°, м№ЁмӢӨк°ҜмҲҳ, лӘҮмёөмқём§Җ, лӘҮл…„лҗҳм—ҲлҠ”м§Җ н•ҙм„ң мҙқ 4к°ңмқҳ featureк°Җ мһҲл„Өмҡ”.
  • $x^i$ : н•ҳлӮҳмқҳ exampleм—җ лҢҖн•ң vectorмһ…лӢҲлӢӨ. мүҪкІҢ л§җн•ҙм„ң н•ҳлӮҳмқҳ rowлҘј лң»н•©лӢҲлӢӨ. $x^3$мқҖ м„ёлІҲм§ё rowмқё [1534 3 2 30] лІЎн„°лҘј к°ҖлҰ¬нӮӨкІҢ лҗ©лӢҲлӢӨ.
  • $x_j^i$ : iлІҲм§ё row dataмқҳ jлІҲм§ё featureмқҳ к°’ мқ„ к°ҖлҰ¬нӮөлӢҲлӢӨ. мқҙлҘјн…Ңл©ҙ $x_4^3$мқҖ 30мқ„ к°ҖлҰ¬нӮӨкІ мЈ ?

Changed Hypothesis

кё°мЎҙмқҳ лӢЁмқјліҖмҲҳ м„ нҳ•нҡҢк·Җм—җм„ңлҠ” лӢӨмқҢкіј к°ҷмқҖ к°Җм„Өн•ЁмҲҳлҘј мқҙмҡ©н–ҲмҠөлӢҲлӢӨ. к·ёлҹ¬лӮҳ, лҚ” л§ҺмқҖ featuresл“Өмқҙ мғқкІЁлӮЁм—җ л”°лқј мқҙ мӢқмқ„ ліҖкІҪмӢңнӮ¬ н•„мҡ”м„ұмқҙ мғқк№ҒлӢҲлӢӨ. кІ°кіјм ҒмңјлЎң мҡ°лҰ¬лҠ” лӢӨмқҢкіј к°ҷмқҖ к°Җм„Өн•ЁмҲҳлҘј м„ёмҡ°кІҢ лҗ©лӢҲлӢӨ.

м—¬кё°м„ң $x_0$мқ„ 1лЎң м„Өм •н•ҳкІҢ лҗңлӢӨл©ҙ, к°Җ мң„мқҳ мӢқкіј лҸҷмқјн•ҙ ліҙмқҙкІҢ лҗ  кІғмһ…лӢҲлӢӨ. лӢӨл§Ң, $x_0$лқјлҠ” featureк°Җ м¶”к°ҖлЎң мғқкІЁлӮҳкІҢ лҗҳл©ҙм„ң, feature vectorмқҳ dimensionмқҖ nм—җм„ң n+1 лЎң м»Өм§‘лӢҲлӢӨ. parameterмқё $\theta$мқҳ dimensionмқҖ кё°мЎҙкіј к°ҷмқҖ n+1мқҙ лҗ©лӢҲлӢӨ. мқҙлЎңмҚЁ л‘җ column vectorл“Өмқҳ dimensionмқҙ лҸҷмқјн•ҙмЎҢмҠөлӢҲлӢӨ.

лӢӨмӢң мқҙ к°Җм„Өн•ЁмҲҳлҘј ліјк№Ңмҡ”? мқҙкІғмқ„ м„ нҳ•лҢҖмҲҳлҘј мқҙмҡ©н•ҳм—¬ к°„лӢЁн•ҳкІҢ н‘ңнҳ„н•ҳл©ҙ, лЎң н‘ңнҳ„н•  мҲҳ мһҲмҠөлӢҲлӢӨ..!! м—„мІӯлӮң мӢқмқҳ к°„мҶҢнҷ”лҘј м§„н–үн•ң м…ҲмқҙмЈ ! мқҙ л•Ң $h_\theta(x)$мқҳ dimensionмқҖ [1 x n+1] * [n+1 x 1] = [1 x 1]мқҙ лҗ©лӢҲлӢӨ.

Gradient Descent for Multiple Variables

к°Җм„Өн•ЁмҲҳ м„Өм •мқ„ л§Ҳм№ң мҡ°лҰ¬лҠ”, мқҙм ң кІҪмӮ¬н•ҳк°•лІ•(Gradient Descent)лҘј м Ғмҡ©н•ҙліј кІғмһ…лӢҲлӢӨ. мқҙм—җ м•һм„ң лӢӨмқҢкіј к°ҷмқҙ к°ңлі„м Ғмқё $\theta$л“ӨлЎң мқҙлЈЁм–ҙм§„ мҡ°лҰ¬мқҳ cost function Jн•ЁмҲҳ мқҳ parameterмқ„ dimensionмқҙ n+1мқё $\theta$ лІЎн„°лЎң л°”кҫём–ҙмЈјлҸ„лЎқ н•©лӢҲлӢӨ.

л§Ҳм°¬к°Җм§ҖлЎң мҡ°лҰ¬мқҳ gradient descent algorithmлҳҗн•ң parameterмқ„ ліҖкІҪн•ҙмЈјлҸ„лЎқ н•©мӢңлӢӨ.

мқҙм ң мқҙ м•Ңкі лҰ¬мҰҳмқҖ лӢӨмқҢкіј к°ҷмқҙ н‘ңкё°лҗ  мҲҳ мһҲмҠөлӢҲлӢӨ.

мң„ мӢқмқҳ лҒқм—җ $x^{(j)}_i$к°Җ л¶ҷмқҖкІғмқҖ нҺёлҜёл¶„мқҳ кІ°кіјлЎң н•ҙм„қлҗ©лӢҲлӢӨ.

мқҙлЎңмҚЁ gradient descentк°Җ multiple variablesм—җ лҢҖн•ҳм—¬ м–ҙл–»кІҢ лӢ¬лқјм§ҖлҠ”м§Җм—җ лҢҖн•ҙ к°„лӢЁн•ҳкІҢ мӮҙнҺҙліҙм•ҳмҠөлӢҲлӢӨ.

м•„лһҳмқҳ мӢқмқҖ көүмһҘнһҲ мӨ‘мҡ”н•ң м—ӯн• мқ„ н•Ёмқ„ кё°м–өн•ҳл©ҙ мўӢмқ„кІғ к°ҷмҠөлӢҲлӢӨ.

Feature Scaling

н•ҳлӮҳк°Җ м•„лӢҢ multiple variableлҘј к°–лҠ” л¬ём ңм—җм„ңлҠ” мқҙ variableл“Өмқҙ л„Ҳл¬ҙлӮҳ нҒ° range м°Ёмқҙк°Җ лӮҳкІҢлҗҳл©ҙ gradient descentлҘј м§„н–үн•  кІҪмҡ° лӢӨмқҢкіј к°ҷмқҙ м§„н–үн•ҳкІҢ лҗ©лӢҲлӢӨ. мқҙл ҮкІҢ м§„н–үн•ҳкІҢ лҗ  кІҪмҡ°м—җлҠ” к¶Ғк·№м Ғмқё global minimumм—җ лҸ„лӢ¬н•ҳкё°к№Ңм§Җ мҳӨлһң мӢңк°„мқҙ кұёлҰ¬кІҢ лҗ©лӢҲлӢӨ.

л”°лқјм„ң мҡ°лҰ¬лҠ” л№ лҘё global minimumмңјлЎңмқҳ мҲҳл ҙмқ„ мң„н•ҙ variables / features л“Өмқҙ л№„мҠ·н•ң rangeлҘј к°–лҸ„лЎқ лӢӨл“¬м–ҙмЈјлҠ” feature scalingмқҙлқјлҠ” мһ‘м—…мқ„ м§„н–үн•ҳкІҢ лҗ©лӢҲлӢӨ.

feature scalingмқҳ кІ°кіјлҠ” лҶҖлһҚмҠөлӢҲлӢӨ. нғҖмӣҗмқҙм—ҲлҚҳ $J(\theta)$ к·ёлһҳн”„мқҳ мқҙмӢ¬мңЁмқҙ м җм°Ё мһ‘м•„м§Җл©ҙм„ң мӣҗмқҳ нҳ•нғңлҘј лқ„кІҢ лҗ©лӢҲлӢӨ.

к°ҖмһҘ мўӢмқҖ лІ”мЈјлҠ” $ -1 \leq x_(i) \leq 1 $ нҳ№мқҖ $ -0.5 \leq x_(i) \leq 0.5 $ лқјкі  н•©лӢҲлӢӨ.

Mean Normalization

feature scalingмқҳ technique мӨ‘ н•ҳлӮҳмқё mean normalizationмқ„ мқҙмҡ©н•ҳм—¬ rangeл“Өмқ„ мқјм •н•ҳкІҢ нҶөмқјмӢңнӮӨлҠ” л°©лІ•мқ„ ліҙм—¬л“ңлҰ¬кІ мҠөлӢҲлӢӨ.

мң„ мӢқмқ„ мқҙмҡ©н•ҳм—¬ mean normalizationмқ„ м§„н–үн•ҳкІҢ лҗ©лӢҲлӢӨ. к·ёл ҮлӢӨл©ҙ мҡ°лҰ¬к°Җ лӢӨмқҢкіј к°ҷмқҖ л¬ём ңм—җм„ң normalizedлҗң $x_2^{(4)}$лҘј кө¬н•ҳлҠ” кІғмқ„ ліҙм—¬л“ңлҰ¬кІ мҠөлӢҲлӢӨ. мң„ мӢқмқҳ $x_i$м—җ $x_2^{(4)}$лҘј лҢҖмһ…н•ҳкі , $\mu_i$м—җ mean мҰү нҸүк· мқ„ лҢҖмһ…н•ҳкі , $s_i$м—җлҠ” мөңлҢ“к°’м—җм„ң мөңмҶҹк°’мқ„ ләҖ rangeк°’мқ„ лҢҖмһ…н•ҳкұ°лӮҳ н‘ңмӨҖнҺём°ЁлҘј лҢҖмһ…н•ҳм—¬ м§„н–үн•©лӢҲлӢӨ.(л¶„лӘ… range лҢҖмһ…мӢңмҷҖ н‘ңмӨҖнҺём°Ё лҢҖмһ…мӢңмқҳ кІ°кіјлҠ” лӢӨлҰ…лӢҲлӢӨ.)

мң„ л§ҒнҒ¬мқҳ л¬ём ңм—җм„ң $s_i = 8836 вҲ’ 4761 = 4075$мқҙкі , $x_i = 4761$, $\mu_i = 6675.5$мқҙлҜҖлЎң, м •лӢө $x_2^{(4)}$лҠ” -0.47мһ…лӢҲлӢӨ. мқҙ л•Ңм—җлҠ” $s_i$лҘј кө¬н•ҳкё° мң„н•ҳм—¬ rangeлҘј мұ„нғқн•ҳмҳҖмҠөлӢҲлӢӨ.

Plot Debugging

xм¶•м—җ iterationмқҳ нҡҹмҲҳ, yм¶•м—җ $J(\theta)$лҘј plotмңјлЎң к·ёл Өліҙл©ҙ, лӢӨмқҢ к·ёлҰјкіј к°ҷмқҙ лҚ”мқҙмғҒ к°җмҶҢн•ҳм§Җ м•ҠлҠ” кө¬к°„мқҙ мғқк№ҒлӢҲлӢӨ.

л¬јлЎ  yм¶•мқё $J(\theta)$к°Җ н•ӯмғҒ к°җмҶҢн•ҳкІҢлҒ” л„Ҳл¬ҙ нҒ° $\alpha$лҘј мһЎм§Җ м•ҠлҸ„лЎқ мЈјмқҳн•ҙм•јн•©лӢҲлӢӨ. $\alpha$к°Җ л„Ҳл¬ҙ нҒ° кІҪмҡ°м—җлҠ” overshootingмқҙ л°ңмғқн•  мҲҳ мһҲмҠөлӢҲлӢӨ. л”°лқјм„ң м ҒлӢ№н•ң $\alpha$лҘј м„Өм •н•ҙм•јн•©лӢҲлӢӨ. мқҙл•Ңмқҳ нҢҒмқҖ м•ҢнҢҢлҘј 3л°°м”© мҰқк°җмӢңмјңмЈјл©ҙм„ң мЎ°м Ҳн•ҳлҠ” кІғмһ…лӢҲлӢӨ. мҳҲлҘјл“Өм–ҙ 0.01, 0.03, 0.003 л“ұкіј к°ҷмқҙ л§җмһ…лӢҲлӢӨ. мқҙл“Ө 0.01, 0.03, 0.003 л“ұмқҳ alphaм—җ лҢҖн•ң plotл“Өмқ„ л№„көҗн•ҙліҙл©ҙм„ң м Ғм Ҳн•ң alphaлҘј м°ҫм•„лӮҙлҠ” кіјм •мқҙ мӨ‘мҡ”н•ҳкІ мҠөлӢҲлӢӨ.

Automatic Convergence Test

iterationмқҙ мң мқҳлҜён•ң м°ЁмқҙлҘј лҚ”мқҙмғҒ л§Ңл“Өм–ҙлӮҙм§Җ лӘ»н•  л•Ңм—җ iterationмқ„ мӨ‘м§ҖмӢңнӮӨкі  мөңмҶҢк°Җ лҗҳлҠ” $J(\theta)$лҘј кө¬н•ҳлҠ” л°©лІ• лҳҗн•ң мЎҙмһ¬н•©лӢҲлӢӨ. лӢӨл§Ң, мқҙ л°©лІ•мқҳ м•„мү¬мҡҙ м җмқҖ мң мқҳлҜён•ң м°Ёмқҙмқё thresholdлҘј м°ҫм•„лӮҙлҠ”кІғмқҙ мүҪм§Җ м•ҠлӢӨлҠ” кІғмһ…лӢҲлӢӨ. мқҙ thresholdк°Җ м Ғм Ҳн•ҳм§Җ м•Ҡмңјл©ҙ, н•ңм°ё лӮҙл Өк°Җкі  мһҲмқ„ л•Ңм—җ iterationмқ„ мӨ‘м§ҖмӢңнӮӨкІҢ лҗ  мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ. л”°лқјм„ң plotмқ„ нҷңмҡ©н•ҳм—¬ xк°’мқҙ мқјм§Ғм„ мқҙ мқҙлЈЁм–ҙм§ҖлҠ” $J(\theta)$лҘј м°ҫм•„лӮҙлҠ” л°©лІ•мқҙ ліҙнҺём Ғмһ…лӢҲлӢӨ.

Choice of Features and Polynomial Regression

Choosing the most appropriate Feature

мҡ°лҰ¬к°Җ м§‘к°’мқ„ мҳҲмёЎн•Ём—җ мһҲм–ҙм„ң л¶Ҳн•„мҡ”н•ң featureлҘј мӮ¬мҡ©н•ҳкі  мһҲмқ„м§ҖлҸ„ лӘЁлҰ…лӢҲлӢӨ. мҳҲлҘјл“Өм–ҙ м§‘мқҳ л„“мқҙлҘј мӨ‘мҡ”н•ң featureлЎң мғқк°Ғн•ҳл©° м§‘мқ„ кө¬н•ҳкі мһҲлҠ”лҚ°, мЈјм–ҙм§„ featureмқҙ кі мһ‘ к°ҖлЎң($x_1$)мҷҖ м„ёлЎң($x_2$) кёёмқҙ(length)лқјл©ҙ, мғҲлЎңмҡҙ featureмқ„ л§Ңл“Өм–ҙм„ң мӮ¬мҡ©н•  мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ. мғҲлЎңмҡҙ featureлҠ” $x_1 * x_2$(к°ҖлЎң * м„ёлЎң)мқҙкІ мЈ . мқҙл ҮкІҢ н•ҳл©ҙ 2к°ңмқҳ featureмқ„ 1к°ңлЎң мӨ„мқј мҲҳ мһҲкІҢлҗ©лӢҲлӢӨ. к·ёлҰ¬кі  мқҙлҠ” лӢӨмқҢкіј к°ҷмқҖ к°Җм„Өн•ЁмҲҳлҘј м„Өм •н•  мҲҳ мһҲм–ҙліҙмһ…лӢҲлӢӨ. мқҙлҘј нҶөн•ҙ м•Ң мҲҳ мһҲл“Ҝмқҙ, мғҲлЎңмҡҙ featureмқ„ л§Ңл“Өм–ҙм„ң мӮ¬мҡ©н•ҳкІҢлҗҳл©ҙ мўӢмқҖ лӘЁлҚёмқ„ л§Ңл“Өм–ҙ лӮј мҲҳ мһҲмҠөлӢҲлӢӨ.

Polynomial Regression

мІҳмқҢм—җ polynomial regressionмқ„ linear regressionм—җм„ң м ‘н•ҳмҳҖмқ„ л•Ңм—җ л¬ёл“қ л¬ҙм–ёк°Җк°Җ мқҙмғҒн•ҳлӢӨлҠ” мғқк°Ғмқҙ л“Өм—ҲмҠөлӢҲлӢӨ. м„ нҳ•нҡҢк·ҖлҘј н•ҳлҠ”лҚ°, polynomial regressionмқҙ к°‘мһҗкё° нҠҖм–ҙлӮҳмҷҖ лӢ№нҷ©н–Ҳм—ҲмҠөлӢҲлӢӨ. м•„лһҳ л§ҒнҒ¬лҠ” linear regressionмқҙ parameterм—җ мһҲм–ҙ linearн•Ёмқ„ м„ӨлӘ…н•ҙмӨҚлӢҲлӢӨ. м°ёкі н•ҙмЈјм„ёмҡ”.

++ why polynomial regression is in the category of linear regression

мҡ°лҰ¬к°Җ linear regressionмңјлЎң лҚ°мқҙн„°лҘј fittingн•ҳлҠ”лҚ°м—җ н•ңкі„лҘј лҠҗлӮ„л•Ңм—җлҠ” polynomial regressionмқ„ мқҙмҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙ л•Ң мҡ°лҰ¬мқҳ к°Җм„Өн•ЁмҲҳ м—җм„ңмқҳ $x_1$мқҖ $x$, $x_2$лҠ” $x^2$, $x_3$мқҖ $x^3$лЎң л§Өн•‘мӢңмјңмЈјл©ҙ лҗ©лӢҲлӢӨ.

мң„мҷҖ к°ҷмқҖ лҚ°мқҙн„°мқҳ кІҪмҡ°м—җлҠ” кјӯм§Җм җ мқҙнӣ„м—җ лӮҙл Өк°ҖлҠ” мҳӨлӘ©н•ң 2м°Ё лӢӨн•ӯмӢқліҙлӢӨлҠ” 3м°Ё лӢӨн•ӯмӢқмқҙ лҚ” мһҳ л“Өм–ҙл§һлҠ”кІғк°ҷмЈ ? 3м°ЁлӢӨн•ӯмӢқ мқҙмҷём—җлҸ„ м ңкіұк·ј н•ЁмҲҳ $\sqrt{x}$($x^{1\over2}$)лҘј мқҙмҡ©н•ҳл©ҙ лҚ” м Ғм Ҳн•ң лҚ°мқҙн„° н”јнҢ…мқҙ лҗ  кІғ к°ҷмҠөлӢҲлӢӨ.

Normal Equation

м—¬нғңк№Ңм§Җ мҡ°лҰ¬лҠ” linear regressionл¬ём ңл“Өмқ„ н•ҙкІ°н•ҳкё° мң„н•ҙ gradient descentлҘј мқҙмҡ©н•ҳм—¬ лҚ°мқҙн„°м—җ к°ҖмһҘ мһҳ л“Өм–ҙл§һлҠ” к°Җм„Өн•ЁмҲҳлҘј м°ҫм•„лӮҳк°”мҠөлӢҲлӢӨ. н•ҳм§Җл§Ң, gradient descent мҷём—җлҸ„ normal equationмқ„ мқҙмҡ©н•ҳм—¬ лҚ°мқҙн„°лҘј н”јнҢ…н•  мҲҳ мһҲмҠөлӢҲлӢӨ. мӮҙм§қ ліөмһЎн•ҳкёҙ н•ҳм§Җл§Ң, л§Һм§Җ м•ҠмқҖ лҚ°мқҙн„°лҘј лӢӨлЈ°л•Ңм—җлҠ” gradient descentліҙлӢӨ л№ лҘҙкІҢ мӮ°мҲ м ҒмңјлЎң $\theta$к°’л“Өмқ„ кө¬н•ҳкІҢ лҗ©лӢҲлӢӨ.

м •к·ңл°©м •мӢқмқ„ мқҙмҡ©н•ң н•ҙлҘј кө¬н•ҳлҠ” мӢқмқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

мң„ мӢқм—җ лҢҖн•ң мһҗм„ён•ң н’ҖмқҙлҠ” мўӢмқҖ к°•мқҳмҳҒмғҒмңјлЎң лҢҖмІҙн•ҳкІ мҠөлӢҲлӢӨ.

++ м •к·ңл°©м •мӢқмқ„ мқҙмҡ©н•ң н•ҙ кө¬н•ҳкё°

к·ёл ҮлӢӨл©ҙ мқҙм ң м •к·ңл°©м •мӢқмқ„ мқҙмҡ©н•ҳм—¬ мЈјм–ҙм§„ лҚ°мқҙн„°м—җ л§һлҠ” $\theta$к°’мқ„ кө¬н•ҙліҙлҸ„лЎқ н•ҳкІ мҠөлӢҲлӢӨ. мҡ°м„ , лҚ°мқҙн„°лҠ” мқҙлІҲ нҸ¬мҠӨнҠё мӢңмһ‘н•  л•Ң мӮ¬мҡ©лҗҳм—ҲлҚҳ multiple featuresк°Җ мһҲлҠ” м§‘к°’мҳҲмёЎ лҚ°мқҙн„°мһ…лӢҲлӢӨ.

м •к·ңл°©м •мӢқм—җм„ңмқҳ XлҠ” design matrixлЎңмҚЁ, лӘЁл“  featureм •ліҙлҘј [m x n+1] matrixм—җ лӢҙмҠөлӢҲлӢӨ. n+1мқҖ $x_0$ featureмқҳ columnмқ„ м¶”к°Җн•ң кІғ мһ…лӢҲлӢӨ.л§Ҳм°¬к°Җм§ҖлЎң, м •к·ңл°©м •мӢқм—җм„ңмқҳ yлҠ” [m x 1] matrixмқҳ column vectorлЎң м •мқҳлҗ©лӢҲлӢӨ.

мң„мқҳ лҚ°мқҙн„°лҘј мқҙмҡ©н•ҳм—¬ м •к·ңл°©м •мӢқм—җ лҢҖмһ…н•ҳкІҢ лҗҳл©ҙ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

мң„ мӢқмқ„ кі„мӮ°н•ҳл©ҙ cost functionмқ„ мөңмҶҢнҷ”н•ҳлҠ”, мҰү лҚ°мқҙн„°м—җ к°ҖмһҘ м Ғн•©н•ң $\theta$к°’мқ„ кө¬н•ҳкІҢ лҗ©лӢҲлӢӨ. н•ңлІҲ кі„мӮ°н•ҙліјк№Ңмҡ”?

к·ёл ҮлӢӨл©ҙ, м–ём ң м–ҙл–Ө кІҪмҡ°м—җ gradient descentлҘј м“°кі , м–ҙл–Ёл•Ң normal equationмқ„ мҚЁм•јн• к№Ңмҡ”? лЁјм Җ Gradient Descentмқҳ мһҘлӢЁм җмқ„ лӮҳм—ҙн•ҙліҙкІ мҠөлӢҲлӢӨ.

  • мһҘм җ
    • м—„мІӯлӮң м–‘мқҳ лҚ°мқҙн„°лҘј лӢӨлЈ° л•Ң л№ лҘҙлӢӨ.
      • м—„мІӯлӮң м–‘мқҙлқј н•ЁмқҖ nмқҙ 10000ліҙлӢӨ нҒ° кІҪмҡ°лҘј лң»н•ңлӢӨ.
  • лӢЁм җ
    • learning rate $\alpha$лҘј м •н•ҙм•јн•ңлӢӨ.
    • iterationмқ„ мӮ¬мҡ©н•ҳкё° л•Ңл¬ём—җ мҶҚлҸ„к°Җ лҠҗлҰ¬лӢӨ.($O(kn^2)$)

л°ҳл©ҙм—җ, Normal Equationмқҳ мһҘлӢЁм җмқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

  • мһҘм җ
    • learning rate $\alpha$лҘј м •н•  н•„мҡ”к°Җ м—Ҷмңјл©°, convergenceлҘј мІҙнҒ¬н•  н•„мҡ”к°Җ м—ҶлӢӨ.
    • iterationмқ„ мӮ¬мҡ©н•ҳм§Җ м•Ҡкё° л•Ңл¬ём—җ мҶҚлҸ„к°Җ л№ лҘё нҺёмқҙлӢӨ.
  • лӢЁм җ
    • м—ӯн–үл ¬мқ„ м—°мӮ°н•  н•„мҡ”к°Җ мһҲмңјл©°, мқҙлҠ” $O(n^3)$мқҳ мӢңк°„ліөмһЎлҸ„лҘј к°–лҠ”лӢӨ.
    • nмқҙ м»Өм§ҲмҲҳлЎқ м—°мӮ°мҶҚлҸ„к°Җ лҠҗл Өм§„лӢӨ.

л”°лқјм„ң, ліҙнҶө nмқҙ 10000мқ„ л„ҳм§Җ м•ҠлҠ” кІҪмҡ°м—җлҠ” к°„лӢЁн•ҳкІҢ normal equationмқ„ мқҙмҡ©н•ҳм—¬ cost functionмқ„ мөңмҶҢнҷ”н•ҳлҠ” $\theta$к°’мқ„ кө¬н•ҳкі , nмқҙ 10000ліҙлӢӨ нҒ° м—„мІӯлӮң м–‘мқҳ лҚ°мқҙн„°мқҳ кІҪмҡ°м—җлҠ” gradient descentлҘј мқҙмҡ©н•ҳм—¬ м Ғм Ҳн•ң $\theta$к°’мқ„ кө¬н•©лӢҲлӢӨ.

Normal Equation and non-invertibility

м„ нҳ•лҢҖмҲҳлҘј кіөл¶Җн•ҳм…Ёкё° л•Ңл¬ём—җ м •к·ңл°©м •мӢқм—җм„ң м“°мқҙлҠ” м—ӯн–үл ¬мқҙ мЎҙмһ¬н•ҳм§Җ м•Ҡмңјл©ҙ м–ҙм©Ңм§ҖлқјлҠ” мғқк°Ғмқ„ н•ҳм…Ёмқ„ мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ. лӢӨн–үмқҙлҸ„, MATLABмқҙлӮҳ octaveмқҳ кІҪмҡ° pinv лқјлҠ” pseudo inverse functionмқҙ мЎҙмһ¬н•ҳм—¬ non-invertibleн•ң matrixм—җ лҢҖн•ҙм„ңлҸ„ мҳ¬л°”лҘё к°’мқ„ кі„мӮ°н•ҙмӨҚлӢҲлӢӨ.

к·ёл ҮлӢӨл©ҙ л§Ҳм§Җл§үмңјлЎң $X^TX$к°Җ non-invertibleн•ҳлӢӨлҠ” кІғмқҙ к°–лҠ” мқҳлҜём—җ лҢҖн•ҙ м•Ңм•„ліҙкІ мҠөлӢҲлӢӨ. мқјл°ҳм ҒмңјлЎңлҠ” л‘җ к°Җм§Җ мӣҗмқёмқҙ мһҲмҠөлӢҲлӢӨ.

мІ«м§ёлҠ” л¶Ҳн•„мҡ”н•ң featureл“Өмқҙ мЎҙмһ¬н•  кІҪмҡ°мһ…лӢҲлӢӨ. л‘җлІҲм§ёлҠ” л„Ҳл¬ҙ л§ҺмқҖ featureл“Өмқҙ мЎҙмһ¬н•  кІҪмҡ°мһ…лӢҲлӢӨ. мҳҲлҘјл“Өм–ҙ, featureмқҳ к°ңмҲҳ nмқҙ mліҙлӢӨ нӣЁм”¬ л§ҺмқҖ кІҪмҡ°м—җ $X^TX$к°Җ к°ҖлҒ” л№„к°Җм—ӯм„ұмқё кІҪмҡ°к°Җ мғқкё°кІҢлҗ©лӢҲлӢӨ. л„Ҳл¬ҙ л§ҺмқҖ featureл“ӨмқҖ м¶”нӣ„м—җ л°°мҡё regularizationмқ„ мқҙмҡ©н•ҳм—¬ л¬ём ңлҘј н•ҙкІ°н•ҳкұ°лӮҳ, featureл“Өмқ„ м§ҖмӣҢлӮҳк°Җл©ҙм„ң н•ҙкІ°н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

лҳҗн•ң, linearly dependentн•ң featureмқ„ м§Җмҡ°лҠ” л°©лІ•лҸ„ мһҲлӢӨкі  н•©лӢҲлӢӨ.

м§ҖкёҲк№Ңм§Җ Linear Regression with Multiple Variables м—җ лҢҖн•ҙ м•Ңм•„ліҙм•ҳмҠөлӢҲлӢӨ. лӢӨмқҢ мӢңк°„м—җлҠ” Octave / Matlab Tutorialмқ„ кұҙл„Ҳлӣ°кі  Logistic Regressionм—җ лҢҖн•ҙ нҸ¬мҠӨнҢ…н•ҳкІ мҠөлӢҲлӢӨ. мҳӨнғҲмһҗ / мҳӨлҘҳ / к¶ҒкёҲмӮ¬н•ӯ лҢ“кёҖлЎң лӮЁкІЁмЈјмӢңл©ҙ м Ғк·№ л°ҳмҳҒн•ҳм—¬ лҚ”мҡұ л°ңм „лҗң лӘЁмҠөмқ„ ліҙм—¬л“ңлҰ¬кІ мҠөлӢҲлӢӨ. к¶ҒкёҲн•ҳмӢ  кІғмқҙ мһҲмңјмӢңл©ҙ м–ём ңл“ м§Җ лҢ“кёҖмқ„ лӮЁкІЁмЈјмӢңкё° л°”лһҚлӢҲлӢӨ. к°җмӮ¬н•©лӢҲлӢӨ.
