DFS ВіцьїевІЮ ьіИвдгВЎђ Ж░ёВёаВЮў вХёвЦў - DFS Spanning Tree and Classifying Edges

DFS ВіцьїевІЮ ьіИвдг

DFS ВіцьїевІЮ ьіИвдгвъђ?

  • DFS ВіцьїевІЮ ьіИвдгвіћ ЖиИвъўьћёВЌљВёю DFSвЦ╝ Вќ┤віљ ВаЋВаљ uВЌљ вїђьЋ┤ ВІцьќЅьЋўВўђВЮё вЋї, uвЦ╝ вБеьіИвАю ьЋўвіћ ьіИвдгвЦ╝ вДљьЋювІц.

Ж░ёВёа вХёвЦўВЮў ВЮ┤ВџЕ

  • ЖиИвъўьћёВЌљВёю Ж░ёВёаВЮё вХёвЦўьЋўЖ▓ївљўвЕ┤ вІцВќЉьЋю ВЋїЖ│авдгВдўВЌљ ВЮЉВџЕьЋа Вѕў ВъѕЖ▓ївљювІц.

  • Ж░ёВёа вХёвЦўвЦ╝ ьєхьЋ┤ ЖиИвъўьћё ВЋїЖ│авдгВдўВЮё ВЮ┤ьЋ┤ьЋўЖ│а ВдЮвфЁьЋўЖИ░ ВюёьЋю вЈёЖхгвАю ВюаВџЕьЋўЖ▓ї ВЊ░ВЮИвІц.

  • ВўѕвЦ╝ вЊцВќ┤, DFSЖ░ђ ВЌГьќЅ Ж░ёВёаВЮё вДївЊцВДђ ВЋіВю╝вЕ┤ ЖиИвъўьћёвіћ ВѓгВЮ┤ьЂ┤ВЮ┤ ВА┤ВъгьЋўВДђ ВЋіЖ│а, ВЌГвЈё Вё▒вдйьЋювІц. ВдЅ, ВѓгВЮ┤ьЂ┤ВЮў ВА┤Въг ВЌгвХђвіћ ВЌГв░ЕьќЦ Ж░ёВёаВЮў ВА┤Въг ВЌгвХђВЎђ вЈЎВ╣ўВЮ┤вІц.

    • ВѓгВЮ┤ьЂ┤ ВА┤Въг ВЌгвХђ ьЎЋВЮИьЋўЖИ░

ЖиИвъўьћёВЮў Ж░ёВёаВЮў ВбЁвЦў

Ж░ёВёаВЮђ ьіИвдг Ж░ёВёа, ВЌГьќЅ Ж░ёВёа, ВѕюьќЅ Ж░ёВёа, ЖхљВ░е Ж░ёВёа В┤Ю 4Ж░ювАю вХёвЦўвљювІц. ВЌГьќЅ Ж░ёВёаЖ│╝ ВѕюьќЅ Ж░ёВёаВЮђ Ж░юВЮИвДѕвІц ВЌГв░ЕьќЦ, Вѕюв░ЕьќЦ Ж░ёВёаВю╝вАю вХђвЦ┤ЖИ░вЈё ьЋювІц.

  • ьіИвдг Ж░ёВёа(Tree edge)
    • ВаЋВаљ vЖ░ђ Ж░ёВёа (u, v)вЦ╝ ьєхьЋ┤ В▓ўВЮї в░юЖ▓гвљўВЌѕвІцвЕ┤, (u, v)віћ ьіИвдгЖ░ёВёаВЮ┤вІц.
    • Ж░ёвІеьЋўЖ▓ївіћ, ВіцьїевІЮ ьіИвдгВЌљ ьЈгьЋевљю Ж░ёВёаВЮё вДљьЋювІц.
  • ВЌГьќЅ Ж░ёВёа(Back edge)
    • ВіцьїевІЮ ьіИвдгВЮў ВъљВєљВЌљВёю ВёаВА░вАю ВЌ░Ж▓░вљўвіћ Ж░ёВёаВЮё вДљьЋювІц.
    • в░ЕьќЦ ЖиИвъўьћёВЌљВёюВЮў ВъљЖИ░ ВѕюьЎў Ж░ёВёавЊцВЮђ ВЌГьќЅ Ж░ёВёаВю╝вАю Ж░ёВБ╝ьЋювІц.
  • ВѕюьќЅ Ж░ёВёа(Forward edge)
    • ьіИвдгЖ░ёВёаВЮ┤ ВЋёвІї ВёаВА░ВЌљВёю ВъљВєљВю╝вАю ВЌ░Ж▓░вљўвіћ Ж░ёВёаВЮё вДљьЋювІц.
  • ЖхљВ░е Ж░ёВёа(Cross edge)
    • Вюё ВёИ вХёвЦўвЦ╝ ВаюВЎИьЋю вѓўвеИВДђ Ж░ёВёаВЮ┤вІц.
    • ВёаВА░ВЎђ ВъљВєљ Ж┤ђЖ│ёЖ░ђ ВЋёвІї(ВюёЖ│ё Ж┤ђЖ│ёЖ░ђ ВЌєвіћ) ВаЋВаљвЊц Ж░ёВЌљ ВЌ░Ж▓░вљю Ж░ёВёавЊцВЮё вДљьЋювІц.
    • ьЋўвѓўВЮў ВаЋВаљВЮ┤ вІцвЦИ ВаЋВаљВЮў ВА░ВЃЂВЮ┤ ВЋёвІѕВќ┤ВЋ╝ ьЋювІц.
    • ВёювАю вІцвЦИ ВіцьїевІЮьіИвдгВЌљ Въѕвіћ вЉљ ВаЋВаљ ВѓгВЮ┤ВЌљ ВъѕВЮёВѕўвЈё ВъѕвІц.

ЖиИвъўьћёВЮў Ж░ёВёаВЮђ DFSвЦ╝ Вќ┤вќц ВаЋВаљВЌљВёю ВІюВъЉьЋўвіћВДђ, Вќ┤вќц ВѕюВёювАю в░ЕвгИьЋаЖ▓ЃВЮИВДђВЌљ вћ░вЮ╝ вІцвЦИ ьіИвдгЖ░ђ ВЃЮВё▒вљўЖ│а, вћ░вЮ╝Вёю Ж░ёВёаВЮў ЖхгвХёВЮ┤ вІгвЮ╝ВДѕ Вѕў ВъѕвІц.

вг┤ьќЦ ЖиИвъўьћёВЌљВёюВЮў Ж░ёВёаВЮў вХёвЦў

  • ЖхљВ░е Ж░ёВёа(Cross edge)Ж░ђ ВА┤ВъгьЋўВДђ ВЋівіћвІц.

  • Вѕюв░ЕьќЦ Ж░ёВёаЖ│╝ ВЌГв░ЕьќЦ Ж░ёВёаВЮў ЖхгвХёВЮ┤ ВЌєвІц.

ВдЅ, вг┤в░ЕьќЦ ЖиИвъўьћёВЮў DFS ВіцьїевІЮ ьіИвдгВЌљВёю вфевЊа Ж░ёВёаВЮђ ьіИвдг Ж░ёВёаВЮ┤Ж▒░вѓў ВЌГьќЅ Ж░ёВёаВЮ┤вІц.

Ж░ёВёаВЮё ЖхгвХёьЋўвіћ DFS ВйћвЊю

  • (u, v)Ж░ђ Вѕюв░ЕьќЦ Ж░ёВёаВЮ┤вЮ╝вЕ┤ vвіћ uВЮў ВъљВєљВЮ┤Вќ┤ВЋ╝ ьЋўЖ│а, вћ░вЮ╝Вёю vЖ░ђ uв│┤вІц вЇћ відЖ▓ї в░юЖ▓гвљўВќ┤ВЋ╝ ьЋювІц. вўљьЋю, (u, v)Ж░ђ ьіИвдг Ж░ёВёаВЮ┤ ВЋёвІѕВќ┤ВЋ╝ ьЋювІц.

  • (u, v)Ж░ђ ВЌГв░ЕьќЦ Ж░ёВёаВЮ┤вЮ╝вЕ┤, vвіћ uВЮў ВёаВА░ВЮ┤Вќ┤ВЋ╝ ьЋювІц. вћ░вЮ╝Вёю vЖ░ђ uв│┤вІц ве╝Вађ в░юЖ▓гвљўВќ┤ВЋ╝ ьЋювІц.

  • (u, v)Ж░ђ ЖхљВ░е Ж░ёВёаВЮ┤вЮ╝вЕ┤, DFS(v)Ж░ђ ВбЁвБївљю ьЏё DFS(u)Ж░ђ ьўИВХювљўВќ┤ВЋ╝ ьЋювІц. вћ░вЮ╝Вёю vвіћ uв│┤вІц ВЮ╝В░Ї в░юЖ▓гвљўВќ┤ВЋ╝ ьЋювІц.

  • ВЮ┤вЦ╝ ВЮ┤ВџЕьЋўВЌг ВйћвЊювЦ╝ ВъЉВё▒ьЋўвЕ┤ вІцВЮїЖ│╝ Ж░ЎвІц.

vector<vector<int>> adj;

vector<int> discovered, finished;

int counter;

void DFS(int node)
{
    discovered[node] = counter++;

    for (int i = 0; i < adj[node].size(); ++i) {
        int next = adj[node][i];
        cout << "(" << node << "," << next << ") : ";

        //  ВЋёВДЂ в░ЕвгИьЋўВДђ ВЋіВЋўвІцвЕ┤
        if (discovered[next] == -1) {
            cout << "tree edge" << endl;
            DFS(next);
        }

        //  nextвЦ╝ в░ЕвгИьќѕВДђвДї ВѕюВёюЖ░ђ nodeв│┤вІц відвІцвЕ┤
        else if (discovered[node] < discovered[next])
            cout << "forward edge" << endl;

        //  nextвЦ╝ в░ЕвгИьќѕВДђвДї ВѕюВёюЖ░ђ nodeв│┤вІц в╣авЦ┤вІцвЕ┤
        //  nextЖ░ђ ВбЁвБїьЋўВДђ ВЋіВЋўвІцвЕ┤ => nextвіћ nodeВЮў ВёаВА░Ж░ђ вље.
        else if (finished[next] == 0)
            cout << "back edge" << endl;

        //  ВЮ┤ ВЎИВЌљвіћ вфевЉљ ЖхљВ░е Ж░ёВёа
        else
            cout << "cross edge" << endl;
    }
    finished[node] = 1;
}

  • Ж░ёВёаВЮў ВаЋВЮўвіћ Ж░ёвІевфЁвБїьЋўВДђвДї, ВЮ┤Ж▓ЃВЮё ВйћвЊювАю ВДЂВаЉВаЂВю╝вАю ЖхгьўёьЋўЖИ░віћ Ж╣ївІцвАГвІц.

  • ЖиИвъўВёю discoveredвЮ╝віћ в░░ВЌ┤ВЌљ counterв│ђВѕўвЦ╝ ВЮ┤ВџЕьЋўВЌг в░юЖ▓г ВѕюВёювЦ╝ ВађВъЦьЋювІц.

  • ВЮ┤ в░░ВЌ┤ВЮђ ВаЋВаљВЮ┤ в░ЕвгИвљўВЌѕвіћВДђ ВЌгвХђвЈё ьЋюв▓ѕВЌљ ВЋївацВцђвІц. ЖИ░ВА┤ DFSВЌљВёюВЮў visited в░░ВЌ┤Ж│╝ ВюаВѓгьЋўвІц.

  • ВЮ┤ в░░ВЌ┤ВЮё ьєхьЋ┤ ВёаВА░ВЎђ ВъљВєљ, ьў╣ВЮђ ВЋёвг┤Ж▓ЃвЈё ВЋёвІї Ж┤ђЖ│ёвЦ╝ в░Юьўђвѓ╝ Вѕў ВъѕвІц.

  • вўљьЋю, ВаЋВаљ uВЌљВёюВЮў ВЮИВаЉьЋю вфевЊа ВаЋВаљВю╝вАюВЮў DFSЖ░ђ ВбЁвБївљўВЌѕвіћВДђвЦ╝ ВађВъЦьЋўвіћ в░░ВЌ┤ finishedвЦ╝ ВЮ┤ВџЕьЋювІц.

В░ИЖ│а ВѓгьЋГ

  • ВѓгвъївЊцВЌљ вћ░вЮ╝ вІцВЮїЖ│╝ Ж░ЎВЮ┤ Ж░ёВёаВЮё ВЃЅЖ╣ћвАю вХёвЦўьЋўвіћ Ж▓йВџ░вЈё ВъѕвІц.
