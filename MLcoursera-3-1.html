ML Coursera 3-1 Logistic Regression

Coursera Andrew Ng көҗмҲҳлӢҳмқҳ ML к°•мқҳлҘј м •лҰ¬н•ң нҸ¬мҠӨнҠёмһ…лӢҲлӢӨ.

Introductionм—җм„ң м–ёкёүн–Ҳл“Ҝмқҙ м§ҖлҸ„н•ҷмҠөмқҖ нҒ¬кІҢ нҡҢк·Җ(Regression)мҷҖ л¶„лҘҳ(Classification)лЎң лӮҳлҲҢ мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙм „к№Ңм§Җ мҡ°лҰ¬лҠ” нҡҢк·Җм—җ лҢҖн•ҙ к°„лӢЁнһҲ м•Ңм•„ліҙм•ҳмҠөлӢҲлӢӨ. лӢӨмқҢмңјлЎң л¶„лҘҳм—җ лҢҖн•ҙ м•Ңм•„ліј м°ЁлЎҖмһ…лӢҲлӢӨ.

Logistic Regression

мҡ°лҰ¬лҠ” мқҙм ң Classification м•Ңкі лҰ¬мҰҳмқё logistic regressionм—җ лҢҖн•ҙ мӮҙнҺҙліҙкІ мҠөлӢҲлӢӨ. лӢӨл§Ң мқҙм—җм•һм„ң, л§ҺмқҖ л¶„л“Өмқҙ мҷң Classificationмқ„ кіөл¶Җн•ҳлҠ”лҚ° regressionмқ„ л°°мҡ°лҠ”м§Җм—җ лҢҖн•ҙ к¶ҒкёҲн•ҳмӢӨ мҲҳ мһҲлӢӨкі  мғқк°Ғн•©лӢҲлӢӨ. л¬јлЎ  м ҖлҸ„ к·ёлһ¬м—Ҳкі мҡ”. вҖңlogistic REGRESSIONвҖңмқҙлқјкі  мқҙлҰ„мқҙ л¶ҷм—¬м§„ кІғмқҖ м—ӯмӮ¬мғҒмқҳ мқҙмң лқјкі  н•©лӢҲлӢӨ. л”°лқјм„ң мқҙлҠ” кё°ліём ҒмңјлЎң л¶„лҘҳлҘј мң„н•ң м•Ңкі лҰ¬мҰҳмқҙл©°, нҡҢк·ҖлқјлҠ” мқҙлҰ„мқҙ мһҲлӢӨкі  н•ҙм„ң нҡҢк·Җлқјкі  мғқк°Ғн•ҳл©ҙ м•Ҳлҗ  кІғ к°ҷмҠөлӢҲлӢӨ. мқҙм—җ лҢҖн•ң ліҙлӢӨ мһҗм„ён•ң м„ӨлӘ…мқҖ м•„лһҳ л§ҒнҒ¬м—җ лӮҳмҷҖмһҲмҠөлӢҲлӢӨ.

++ Why is logistic regression called regression

л¶„лҘҳм—җм„ң кё°ліём ҒмңјлЎң yлҠ” discreteн•ң к°’мқҙм–ҙм•ј н•©лӢҲлӢӨ. мҳҲлҘј л“Өм–ҙ, м–ҙл–Ө мқҙл©”мқјмқҙ мҠӨнҢёмқём§Җ(1) м•„лӢҢм§Җ(0), нҳ№мқҖ м–ҙл–Ө мў…м–‘мқҙ м•…м„ұмқём§Җ(1) м–‘м„ұмқём§Җ(0)мҷҖ к°ҷмқҙ 0, 1, 2, 3, вҖҰ (нҳ№мқҖ 1, 2, 3, 4, вҖҰл“ұ мқҙмӮ°м Ғмқё к°’л“Өмқ„ yм—җ мӮ¬мҡ©н•©лӢҲлӢӨ.

лЁјм Җ, binary class classification problemмқ„ мӮҙнҺҙліҙкІ мҠөлӢҲлӢӨ. мӮ¬мӢӨ binaryмҷҖ multiclass classification problemмқҖ кұ°мқҳ л№„мҠ·н•©лӢҲлӢӨ. лӢӨл§Ң, л§Ҳм§Җл§үм—җ one vs all classificationмқ„ м Ғмҡ©н•ҳкІҢ лҗҳлҠ”лҚ°, мқҙлҠ” л’Өм—җм„ң лӢӨмӢң мӮҙн”јлҸ„лЎқ н•ҳкІ мҠөлӢҲлӢӨ.

linear regression to a Classification Problem?

м„ нҳ•нҡҢк·Җм—җ л¶„лҘҳлҘј м Ғмҡ©н•ҳл Өл©ҙ, мқјм • к°’ мқҙмғҒмқҙл©ҙ 1, лҜёл§Ңмқҙл©ҙ 0 л“ұмқҳ л¶„лҘҳ мһ„кі„к°’мқҙ н•„мҡ”н•ңлҚ°, мқҙлҘј thresholdлқјкі  н•©лӢҲлӢӨ. мң„ к·ёлҰјм—җм„ңлҠ” thresholdк°Җ 0.5мһ„мқ„ м•Ң мҲҳ мһҲмҠөлӢҲлӢӨ.

мң„ к·ёлҰјмІҳлҹј мқҙм „м—җ кіөл¶Җн•ң м„ нҳ•нҡҢк·ҖлҘј л¶„лҘҳм—җлҸ„ м Ғмҡ©мӢңнӮ¬ мҲҳ мһҲлӢӨл©ҙ л„Ҳл¬ҙ нҺён• кІғ к°ҷмҠөлӢҲлӢӨ. н•ҳм§Җл§Ң м•„мүҪкІҢлҸ„ лӘҮк°Җм§Җ л¬ём ңм җл“Өмқҙ л°ңмғқн•ҳкІҢ лҗ©лӢҲлӢӨ. н•ңлІҲ л¶„лҘҳл¬ём ңм—җ м„ нҳ•нҡҢк·ҖлҘј мӢңлҸ„н•ҙліҙл©ҙм„ң к·ё л¬ём ңм җл“Өмқ„ нҢҢм•…н•ҙліҙкІ мҠөлӢҲлӢӨ.

    1. мң„ к·ёлҰјл§Ң ліҙл©ҙ м–јн•Ҹ мһҳ fittingлҗҳлҠ”кІғмІҳлҹј ліҙмқҙкё°лҸ„ н•©лӢҲлӢӨ. н•ҳм§Җл§Ң мң„ лҚ°мқҙн„°м…Ӣм—җ л§Өмҡ° мһ‘мқҖ мў…м–‘ нҒ¬кё°мқҳ Malignantн•ң мў…м–‘мқҙ л°ңкІ¬лҗңлӢӨл©ҙ мһҗм№« л§ҺмқҖ лҚ°мқҙн„°л“Өмқҙ мқҙмғҒн•ҳкІҢ л¶„лҘҳлҗ  мҲҳ мһҲлҠ” мғҒнҷ©мқҙ л°ңмғқн•©лӢҲлӢӨ.
    1. мҡ°лҰ¬к°Җ мҳҲмёЎн•ҳлҠ” yк°’мқҖ 0 нҳ№мқҖ 1 л“ұмқҳ к°’мқҙм§Җл§Ң, linear regressionмқ„ мң„н•ң hypothesis functionмқҙ мЈјлҠ” кІ°кіјк°’мқҖ 0ліҙлӢӨ мһ‘мқ„ мҲҳлҸ„, 1ліҙлӢӨ нҒҙ мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ. л”°лқјм„ң мҡ°лҰ¬лҠ” 0 нҳ№мқҖ 1мқҳ к°’мқ„ к°–лҸ„лЎқ л§Ңл“Өм–ҙмӨ„ н•„мҡ”к°Җ мһҲмқ„ кІғмһ…лӢҲлӢӨ.

Hypothesis Representation

к°Җм„Өн•ЁмҲҳмқҳ мЎ°кұҙ

мҡ°лҰ¬к°Җ л¶„лҘҳм—җ мӮ¬мҡ©н•  к°Җм„Өн•ЁмҲҳ $h_\theta(x)$лҠ” 0кіј 1мӮ¬мқҙмқҳ к°’мқ„ к°Җм§Җл©ҙ мўӢмқ„ кІғ к°ҷмҠөлӢҲлӢӨ. лӢӨмқҢкіј к°ҷмқҙ л§җмһ…лӢҲлӢӨ.

мқҙлҘј л§ҢмЎұн•ҳлҠ” н•ЁмҲҳк°Җ мһҲлҠ”лҚ°, л°”лЎң sigmoid function мһ…лӢҲлӢӨ. мқҙ н•ЁмҲҳмқҳ мӢқмқҖ ${1 \over 1+e^{-x}}$ мһ…лӢҲлӢӨ. DesmosлҘј мқҙмҡ©н•ҳм—¬ к·ёл Өліҙл©ҙ, лӢӨмқҢкіј к°ҷмқҖ к·ёлһҳн”„лҘј лқ•лӢҲлӢӨ.

мҡ°лҰ¬лҠ” мқҙ мӢқмқ„ мқҙмҡ©н•ҳм—¬ к°Җм„Өн•ЁмҲҳлҘј м„ёмҡё кІғмһ…лӢҲлӢӨ. мқҙ л•Ң кё°мЎҙмқҳ м„ нҳ•нҡҢк·Җ к°Җм„Өн•ЁмҲҳ мӢқмқё м—җ gлқјлҠ” н•ЁмҲҳлҘј н•©м„ұн•ң н•ЁмҲҳмқё лқјлҠ” мӢқмқ„ м–»м–ҙм„ң мқҙ мӢқмқҳ $\theta^Tx$лҘј $z$лЎң м№ҳнҷҳн•ҳлҠ” кіјм •мқ„ кұ°м№©лӢҲлӢӨ. к·ё кІ°кіј лӢӨмқҢкіј к°ҷмқҖ н•ЁмҲҳк°Җ лӮҳмҳөлӢҲлӢӨ.

к°Җм„Өн•ЁмҲҳ outputк°’мқҳ н•ҙм„қ

мҡ°лҰ¬к°Җ м„ёмҡҙ к°Җм„Өн•ЁмҲҳ h(x)к°Җ мҲ«мһҗлҘј л°ҳнҷҳн•  л•Ң, мҡ°лҰ¬лҠ” мқҙлҘј н•ҙлӢ№ xк°Җ inputмңјлЎң л“Өм–ҙк°”мқ„ л•Ң y=1мқҙ лҗҳлҠ” нҷ•лҘ мқҙлқј н•©лӢҲлӢӨ. мҳҲлҘјл“Өм–ҙ мЈјм–ҙм§„ x к°’мқҙ м–ҙл–Ө нҷҳмһҗмқҳ мқҳлЈҢлҚ°мқҙн„°(мў…м–‘мқҳ нҒ¬кё°, лӘЁм–‘ л“ұ)лқј н–Ҳмқ„ л•Ң, outputмқҙ h(x) = 0.83 мқҙлқјкі  лӮҳмҳӨкІҢ лҗңлӢӨл©ҙ, мқҙ нҷҳмһҗмқҳ мў…м–‘мқҙ м•…м„ұмқј нҷ•лҘ мқҖ 83%к°Җ лҗҳлҠ” кІғ мһ…лӢҲлӢӨ.

мқҙлҹ¬н•ң кІғмқ„ мӢқмңјлЎң н‘ңнҳ„н•ҳл©ҙ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

лҳҗн•ң, мҡ°лҰ¬к°Җ лӢӨлЈЁлҠ” мқҙм§„л¶„лҘҳ(Binary Classification)м—җм„ң лӢӨмқҢмқҙ м„ұлҰҪн•©лӢҲлӢӨ.

мҡ°лҰ¬к°Җ к°–кІҢ лҗ  yк°’мқҖ 0 м•„лӢҲл©ҙ 1мқҙкё° л•Ңл¬ёмқҙмЈ .

Decision Boundary

мҡ°лҰ¬мқҳ к°Җм„Өн•ЁмҲҳ h(x) = g(z) лҠ” мӢңк·ёлӘЁмқҙл“ң н•ЁмҲҳ мһ…лӢҲлӢӨ. h(x)к°Җ 0.5 мқҙмғҒмқј кІҪмҡ° y=1, 0.5 лҜёл§Ңмқј кІҪмҡ° y=0 мқҙлқјкі  л¶„лҘҳлҘј н•ҳкІҢ лҗҳлҠ” кІғмһ…лӢҲлӢӨ. мқҙ л•Ң, g(z)к°Җ 0.5 мқҙмғҒмқё л¶Җл¶„мқҖ к·ёлһҳн”„мғҒм—җм„ң нҷ•мқё к°ҖлҠҘн•ҳл“Ҝмқҙ zк°Җ м–‘мҲҳмқё кІҪмҡ°мһ…лӢҲлӢӨ. мҡ°лҰ¬лҠ” zлҘј $\theta^Tx$лЎң м •мқҳн–ҲмңјлҜҖлЎң, лӢӨмқҢкіј к°ҷмқ„ кІҪмҡ°м—җ h(x)к°Җ 0.5ліҙлӢӨ м»Өм§Җл©°, мқҙлҠ” y=1мқҙлқјлҠ” outputмқ„ л§Ңл“Өм–ҙлғ…лӢҲлӢӨ.

if $ \theta^Tx \ge 0 $ then $h_\theta(x) \ge 0.5,$ $y=1$

л”°лқјм„ң, $ \theta^Tx \ge 0.5 $ мқј л•Ң $y = 1$ мқҙ м„ұлҰҪн•©лӢҲлӢӨ. мҡ°лҰ¬лҠ” мқҙлҘј мқҙмҡ©н•ҙ Decision BoundaryлқјлҠ” кІҪкі„лҘј л§Ңл“Өм–ҙлӮј мҲҳ мһҲмҠөлӢҲлӢӨ.

м¶”к°ҖлЎң, g(z)к°Җ 0.5мқё кІҪмҡ°м—җлҠ” мЈјлЎң y=1лЎң нҢҗм •н•ҳкё°лҠ” н•ҳм§Җл§Ң, к·ё нҢҗм •мқҙ нӢҖлҰҙнҷ•лҘ мқҙ лҶ’лӢӨлҠ” кІғмқ„ м—јл‘җн•ҙм•ј н•  кІғ к°ҷмҠөлӢҲлӢӨ.

ліёкІ©м ҒмңјлЎң, лӢӨмқҢкіј к°ҷмқҖ к°Җм„Өн•ЁмҲҳм—җ, лҚ°мқҙн„°м…Ӣмқҙ мЈјм–ҙмЎҢмқ„ кІҪмҡ°мқҳ л¶„лҘҳлҘј мғқк°Ғн•ҙлҙ…мӢңлӢӨ.

мң„мҷҖ к°ҷмқҖ лӢЁмҲңн•ң лҚ°мқҙн„°м…Ӣмқ„ л¶„лҘҳн•ҳл Өл©ҙ м–ҙл–Ө л°©лІ•мқҙ мһҲмқ„к№Ңмҡ”? мғҒмӢқм ҒмңјлЎңлҠ”, л‘ҳ мӮ¬мқҙм—җ м§Ғм„ мқ„ кёӢлҠ” л°©лІ•мқҙ мһҲмқ„ кІғ к°ҷмҠөлӢҲлӢӨ.

н•ҳм§Җл§Ң, м»ҙн“Ён„°лҠ” мғҒмӢқмқ„ лӘЁлҰ…лӢҲлӢӨ. л”°лқјм„ң, м§Ғм„ мқ„ кёӢкІҢлҗҳлҠ” мӣҗм№ҷкіј л°©лІ•мқ„ м•Ңл ӨмЈјм–ҙм•ј н•©лӢҲлӢӨ.

h(x)к°Җ y=1 мқҙлқјлҠ” кІ°кіјк°’мқ„ лҸ„м¶ңн•ҙлӮҙлҠ” кіјм •мқҳ мӨ‘мӢ¬м—җлҠ” $\theta^Tx \ge 0$ мқҙ мһҲм—ҲмҠөлӢҲлӢӨ. л”°лқјм„ң, мқҙ $\theta^Tx \ge 0$, мҰү $z$к°Җ 0ліҙлӢӨ нҒ¬кұ°лӮҳ к°ҷмқҖ кІҪмҡ°лҠ” 1, мһ‘мқҖ кІҪмҡ°лҠ” 0мқ„ л¶„лҘҳн•ҙлӮҙлҸ„лЎқ л§Ңл“Өл©ҙ лҗ©лӢҲлӢӨ.

мң„ к°Җм„Өн•ЁмҲҳм—җм„ң $z$лҠ” $\theta_0 + \theta_1 x_1 + \theta_2 x_2$ мһ…лӢҲлӢӨ. мҡ°лҰ¬лҠ” мқҙ $\theta$к°’л“Өм—җ лӢӨмқҢкіј к°ҷмқҖ к°’мқ„ м •н•ҙмӨҳліҙкІ мҠөлӢҲлӢӨ.

мң„ кІҪмҡ° $z$лҠ” y=1мқј л•Ң лӢӨмқҢкіј к°ҷмқҖ л¶Җл“ұмӢқмқҙ м„ұлҰҪн•©лӢҲлӢӨ.

к·ёлҰ¬кі  мң„ мӢқмқ„ м •лҰ¬н•ҳл©ҙ, кі л“ұн•ҷкөҗ мҲҳн•ҷмӢңк°„м—җ л°°мҡҙ лӢЁмҲңн•ң вҖңл¶Җл“ұмӢқмқҳ мҳҒм—ӯвҖқ л¬ём ңк°Җ лҗ©лӢҲлӢӨ. м Җ мҳҒм—ӯмқ„ л§ҢмЎұн•ҳлҠ” xл“Өмқҳ кІҪмҡ° y=1мқҙ лҗҳлҠ”кІғмқҙм§Җмҡ”. к·ёлҰјмңјлЎң к·ёлҰ¬л©ҙ, лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

лӢӨмқҢкіј к°ҷмқҙ л‘җ лҚ°мқҙн„°лҘј к°ҖлҘҙлҠ” м§Ғм„ мқ„ кёӢлҠ” л°©лІ•мқҙ мһҲкІ мЈ . мқҙ л•Ң, мқҙ м§Ғм„ мқҙ л°”лЎң Decision Boundary мһ…лӢҲлӢӨ.

мӢ кё°н•ң м җмқҖ, мқҙ Decision BoundaryлҠ” к°Җм„Өн•ЁмҲҳмҷҖ л§Өк°ңліҖмҲҳл§Ңмқ„ мқҙмҡ©н•ҳм—¬ м„ёмҡҙкІғмқҙл©°, м–ҙл– н•ң лҚ°мқҙн„°лҸ„ мӮ¬мҡ©лҗҳм§Җ м•Ҡм•ҳлӢӨлҠ” м җмһ…лӢҲлӢӨ. мҡ°лҰ¬лҠ” лҚ°мқҙн„°лҘј м Ғм Ҳн•ң л§Өк°ңліҖмҲҳлҘј м°ҫлҠ”лҚ° мӮ¬мҡ©н•ҳкІҢ лҗҳлҠ”лҚ°, мқҙл ҮкІҢ н•ҙм„ң м–»м–ҙм§„ л§Өк°ңліҖмҲҳк°Җ мң„ к·ёлҰјмқҳ кІҪмҡ° $\theta^T = [-3, 1, 1]$ мқҙлқјкі  ліҙмӢңл©ҙ лҗ  кІғ к°ҷмҠөлӢҲлӢӨ.

Non-linear Decision Boundaries

Decision BoundaryлҠ” linearн•ҳм§Җ м•Ҡмқ„мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ. лӢӨмқҢкіј к°ҷмқҖ Decision BoundaryлҸ„ мЎҙмһ¬н•©лӢҲлӢӨ.

мң„ кІҪмҡ°м—җлҠ” к°Җм„Өн•ЁмҲҳ $h_\theta(x)$к°Җ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

мқҙ к°Җм„Өн•ЁмҲҳм—җ лҚ°мқҙн„°м…Ӣмқ„ мқҙмҡ©н•ҳм—¬ к°ҖмһҘ м Ғм Ҳн•ң л§Өк°ңліҖмҲҳлҘј м°ҫм•„ліҙм•ҳлҚ”лӢҲ $\theta^T = [-1, 0, 1, 1]$ мҳҖкі , мқҙлҘј нҶөн•ҙ мҡ°лҰ¬лҠ” $x_1^2 + x_2^2 \ge 1$ мқҙлқјлҠ” л°ҳм§ҖлҰ„мқҙ 1мқё мӣҗмқҳ л°”к№Ҙл¶Җл¶„м—җ н•ҙлӢ№н•ҳлҠ” xк°’л“Өмқҙ y=1мқ„ outputмңјлЎң мӨҖлӢӨлҠ” кІғмқ„ м•Ң мҲҳ мһҲмҠөлӢҲлӢӨ.

м¶”к°Җм ҒмңјлЎң, мӣҗліҙлӢӨ лҚ”мҡұ кі м°Ёмӣҗмқҳ ліөмһЎн•ң decision boundaryлҘј л§Ңл“ңлҠ” кІғ лҳҗн•ң к°ҖлҠҘн•©лӢҲлӢӨ. мқҙ л•ҢлҠ” $z$лҘј ліөмһЎн•ҳкІҢ м„Өм •н•ҙмЈјл©ҙ лҗҳкІ мЈ ?

Cost Function for Logistic Regression

Cost functionмқҳ ліём§ҲмқҖ кІ°көӯ к°ҖмһҘ м Ғм Ҳн•ң л§Өк°ңліҖмҲҳ $\theta$лҘј м°ҫм•„лӮҙлҠ” кІғмһ…лӢҲлӢӨ. мқҙлҠ” costк°Җ к°ҖмһҘ м ҒмқҖ parameterлҘј мұ„нғқн•ҳл©ҙм„ң мқҙлЈЁм–ҙм§‘лӢҲлӢӨ. м•һм„  linear regressionмқҙ к·ёлһ¬л“Ҝмқҙ, logistic regressionм—җм„ңлҸ„ ліём§ҲмқҖ ліҖн•ҳм§Җ м•ҠмҠөлӢҲлӢӨ.

linear regressionм—җм„ңмқҳ cost function $J(\theta)$лҠ” лӢӨмқҢкіј к°ҷм•ҳмҠөлӢҲлӢӨ.

мқҙн•ҙмқҳ нҺёмқҳлҘј мң„н•ҙм„ң logistic regressionм—җм„ңлҠ”, linear regressionмқҳ costмқё $ {1 \over 2} (h_\theta(x^{(i)}) - y^{(i)})^2 $мқ„ нҶөм§ёлЎң $ Cost(h_\theta(x), y) $лЎң л°”кҫём–ҙлҙ…мӢңлӢӨ. Cost functionмқҳ кё°лҠҘ(mк°ң лҚ°мқҙн„°мқҳ costл“Өмқҳ н•©)мқҙ лҳ‘к°ҷмқҙ мң м§ҖлҗҳлҸ„лЎқ л§җмқҙмЈ . JлҘј л°”кҫј кІ°кіј лӢӨмқҢкіј к°ҷмқҖ мӢқмқҙ л§Ңл“Өм–ҙм§‘лӢҲлӢӨ.

мқҙл ҮкІҢк№Ңм§Җ мӢқмқ„ ліҖнҳ•н•ҳлҠ” мқҙмң лҠ”, кё°мЎҙмқҳ Squared Error Functionм—җ logistic regressionмқҳ к°Җм„Өн•ЁмҲҳ h(x) = g(z) лҘј лҢҖмһ…н•ҳл©ҙ л¬ём ңк°Җ мғқкё°кё° л•Ңл¬ёмқёлҚ°мҡ”. linear regressionмқҳ к°Җм„Өн•ЁмҲҳ h(x)мқҳ кІҪмҡ° linearity, мҰү м„ нҳ•мқҙкё° л•Ңл¬ём—җ кІ°кіјм ҒмңјлЎң convexн•ң functionмқҙм—ҲмҠөлӢҲлӢӨ. н•ҳм§Җл§Ң, logistic regressionмқҳ к°Җм„Өн•ЁмҲҳ h(x) = g(z)мқҳ кІҪмҡ°, мӢңк·ёлӘЁмқҙл“ң н•ЁмҲҳ $ h_\theta(x) = g(z) = {1 \over 1+e^{-z}} $к°Җ л№„м„ нҳ•; мҰү м„ нҳ•кІ°н•©мңјлЎң н‘ңнҳ„лҗ  мҲҳ м—Ҷкё° л•Ңл¬ём—җ, non-convexн•ң functionмқҙ лҗ©лӢҲлӢӨ.

кө¬мІҙм ҒмңјлЎң non-convexн•Ёмқ„ нҷ•мқён•ҙліҙл©ҙ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ. лЁјм Җ, к°Җм„Өн•ЁмҲҳ g(z)к°Җ л№„м„ нҳ•мқҙкё° л•Ңл¬ём—җ

мң„ мӢқмқ„ лӢӨмӢң $J(\theta)$м—җ лҢҖмһ…н•ҳм—¬ $\theta$мҷҖ $J(\theta)$м—җ лҢҖн•ң к·ёлһҳн”„лҘј plot н•ҙліҙкІҢ лҗҳл©ҙ лӢӨмқҢкіј к°ҷмқҖ лӘЁм–‘мқҙ лӮҳмҳөлӢҲлӢӨ.

Cost function Jк°Җ мҲҳл§ҺмқҖ local minimumмқ„ к°–кІҢлҗҳл©°, мқҙлҠ” Gradient DescentлҘј мқҙмҡ©н•ҳм—¬ global minimumмқ„ м°ҫлҠ”кІғмқ„ м–ҙл өкІҢн•©лӢҲлӢӨ.

л”°лқјм„ң к°Җм„Өн•ЁмҲҳк°Җ convexн•ңм§Җ м—¬л¶ҖлҠ” көүмһҘнһҲ мӨ‘мҡ”н•©лӢҲлӢӨ. мқҙм—җ л”°лқј мҡ°лҰ¬лҠ” $ Cost(h_\theta(x), y) $лҘј convexн•ң мӢқмңјлЎң ліҖкІҪн•  кІғмһ…лӢҲлӢӨ. мғҲлЎңмҡҙ CostлҠ” лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

мқҙкІғмқҙ мҡ°лҰ¬к°Җ logistic regressionм—җм„ң мӮ¬мҡ©н•  costмқҙкі , л”°лқјм„ң мҡ°лҰ¬мқҳ cost function $J(\theta)$лҠ” лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

м—¬кё°м„ң м¶”к°ҖлЎң yк°Җ 1мқҙл“  0мқҙл“  кҙҖкі„м—Ҷмқҙ к°„мҶҢнҷ”лҗң(simplified) н•ҳлӮҳмқҳ мӢқмңјлЎң лі‘н•©мқ„ н•  мҲҳ мһҲлҠ”лҚ°, лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

мқҙлҠ” yк°Җ л¬ҙмЎ°кұҙ 0 нҳ№мқҖ 1мқҙкё°м—җ к°ҖлҠҘн•ң кІғ мһ…лӢҲлӢӨ.

Cost Function Jм—җ лҢҖн•ң мҶҢк°ңлҠ” л§ҲміӨкі , мқҙ н•ЁмҲҳм—җ Gradient DescentлҘј м Ғмҡ©н•ҳкё° мқҙм „м—җ, Cost Functionм—җ лҢҖн•ҙ м–ҙл–»кІҢ м Җ н•ЁмҲҳк°Җ лӮҳмҷ”лҠ”м§Җм—җ лҢҖн•ң мқҙн•ҙлҘј н•ҳкі  л„ҳм–ҙк°ҖкІ мҠөлӢҲлӢӨ. мқҙкІғмқҖ лӢӨмӢң лӢӨмқҢ мӢқм—җм„ң мӢңмһ‘лҗ©лӢҲлӢӨ.

Costлқј н•ЁмқҖ кё°ліём ҒмңјлЎң мҡ°лҰ¬к°Җ м„ёмҡҙ к°Җм„Өмқҳ кІ°кіјк°’кіј мӢӨм ң кІ°кіјлҚ°мқҙн„°к°’мқҳ м°ЁмқҙлҘј мқҳлҜён•©лӢҲлӢӨ. к°Җм„Өн•ЁмҲҳк°Җ y=0мқ„ мҳҲмёЎн–ҲлҠ”лҚ° y=1мқҙ м •лӢөмқҙм—ҲлӢӨл©ҙ, м—„мІӯлӮҳкІҢ нҒ° л№„мҡ©, мҰү $ \infty $ Cost лҘј мҙҲлһҳн•ҳкІ мЈ . л°ҳлҢҖлЎң, к°Җм„Өн•ЁмҲҳк°Җ y=1мқ„ мҳҲмёЎн–ҲлҠ”лҚ°, м •лӢө м—ӯмӢң мҳҲмёЎн•ңлҢҖлЎң y=1мқҙлқјл©ҙ, 0мқҳ л№„мҡ©мқ„ мҙҲлһҳн•  кІғмһ…лӢҲлӢӨ. мқҙлҹ¬н•ң кІғл“Өмқ„ мһҳ м„ӨлӘ…н•ҙмЈјлҠ” н•ЁмҲҳк°Җ $ -log(h_\theta(x)) $мһ…лӢҲлӢӨ. лӢӨмқҢкіј к°ҷмқҖ y=-log(x)к·ёлһҳн”„лҘј лқ„л©°, л°©кёҲ лӮҙмҡ©мқҙ мүҪкІҢ мқҙн•ҙк°Җ к°Ҳ кІғмһ…лӢҲлӢӨ.

y=0мқјл•Ңмқҳ кІҪмҡ°лҘј мғқк°Ғн•ҙліҙм„ёмҡ”! л№„мҠ·н•ң л°©мӢқмңјлЎң м ‘к·јн•ҳл©ҙ мқҙн•ҙн•ҳмӢӨ мҲҳ мһҲмңјмӢӨ кІғмһ…лӢҲлӢӨ.

Minimizing Cost Function with Gradient Descent

мқҙм ңк»Ҹ кө¬н•ҙмҳЁ Cost Function JлҘј мқҙм ңлҠ” мөңмҶҢнҷ”н•  м°ЁлЎҖмһ…лӢҲлӢӨ.

к°ҖмһҘ м ҒмқҖ costлҘј к°–лҠ” $\theta$лҘј м°ҫмқҖ нӣ„ к°Җм„Өн•ЁмҲҳм—җ н…ҢмҠӨнҠё лҚ°мқҙн„° $x$лҘј л„Јм–ҙм„ң лӮҳмҳӨлҠ” кІ°кіјк°’мқҙ л°”лЎң $h_\theta(x)$мқҙл©° мқҙкІғмқҖ y=1мқј нҷ•лҘ мқ„ лӮҳнғҖлӮҙмӨҚлӢҲлӢӨ.

мқҙм „м—җ linear regressionмқҳ Gradient DescentлҠ” лӢӨмқҢкіј к°ҷм•ҳмҠөлӢҲлӢӨ.

Logistic Regressionмқҳ Gradient Descent лҳҗн•ң мң„мҷҖ к°ҷмҠөлӢҲлӢӨ..! мқҙлҠ” logistic regressionмқҳ cost function $J(\theta)$лҘј нҺёлҜёл¶„н•ҙліҙл©ҙ м•Ң мҲҳ мһҲлҠ”лҚ°мҡ”, лӢӨмқҢкіј к°ҷмқҖ кіјм •мқ„ кұ°м№©лӢҲлӢӨ.

мҡ°лҰ¬лҠ” for-loopмқ„ мқҙмҡ©н•ҳм—¬ $\theta$лҘј updateн•  мҲҳ мһҲм§Җл§Ң, лҚ” лӮҳмқҖ vectorizationмқ„ мқҙмҡ©н•  мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ. лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

Advanced Optimization

Gradient DescentлҘј кө¬нҳ„н•  л•Ңм—җлҠ” лӢӨмқҢкіј к°ҷмқҖ кіјм •мқ„ кұ°м№©лӢҲлӢӨ.

  1. $J(\theta)$мҷҖ к·ё нҺёлҜёл¶„мқ„ кө¬нҳ„н•ңлӢӨ.
  2. мң„ л‘ҳмқ„ gradient descentмӢқм—җ лҢҖмһ…н•ңлӢӨ.

мқҙлҹ° Gradient descentліҙлӢӨ н–ҘмғҒлҗң м•Ңкі лҰ¬мҰҳмқҙ лӘҮ мһҲмҠөлӢҲлӢӨ. л°”лЎң Confugate gradient, BFGS, L-BFGSмһ…лӢҲлӢӨ. мқҙл“Өмқҳ мһҘм җмқҖ learning rate $\alpha$лҘј мһҗлҸҷмңјлЎң мһЎм•„мЈјкі  мҶҚлҸ„лҸ„ кё°мЎҙмқҳ кІҪмӮ¬н•ҳк°•лІ•ліҙлӢӨ л№ лҘҙлӢӨлҠ” кІғмһ…лӢҲлӢӨ. к·ёлҹ¬н•ң мқҙмң лЎң м»ӨлӢӨлһҖ лЁёмӢ лҹ¬лӢқ л¬ём ң(huge feature set)л“Өм—җ мһҗмЈј м“°мһ…лӢҲлӢӨ.

Multiclass Classification Problems

м§ҖкёҲк№Ңм§Җ л‘җ к°ңмқҳ classл§Ңмқҙ мЎҙмһ¬н•ҳлҠ” мқҙм§„л¶„лҘҳ(Binary Classification)мқ„ н•ҙмҷ”лӢӨл©ҙ, мӢӨм ңлЎң л§Һмқҙ м“°мқј лӢӨмӨ‘л¶„лҘҳм—җ лҢҖн•ҙ м•Ңм•„ліҙкІ мҠөлӢҲлӢӨ. лӢӨмӨ‘л¶„лҘҳлҠ” мқҙм§„л¶„лҘҳмқҳ м—°мһҘм„ мқҙл©°, л§Ё мң—л¶Җл¶„м—җм„ң мһ к№җ м–ёкёүн•ң one vs all classificationмқ„ мқҙмҡ©н•©лӢҲлӢӨ.

One vs All Classification

л°©лІ•мқҖ к°„лӢЁн•©лӢҲлӢӨ. к°Ғ нҒҙлһҳмҠӨл§ҲлӢӨ мқҙм§„л¶„лҘҳлҘј м§„н–үн•©лӢҲлӢӨ. лӢӨмқҢкіј к°ҷмқҙ л§җмһ…лӢҲлӢӨ.

мғҲлЎңмҡҙ лҚ°мқҙн„° $x$лҘј л°ӣм•„ $h_\theta(x)^{(i)}$к°Җ мөңлҢҖк°Җ лҗҳлҠ” i нҒҙлһҳмҠӨлҘј м°ҫмҠөлӢҲлӢӨ. к·ё i нҒҙлһҳмҠӨлҘј мұ„нғқн•©лӢҲлӢӨ.

кө¬мІҙм ҒмңјлЎңлҠ”, к°Ғк°Ғ н•ҷмҠөн•ң м„ё к°ңмқҳ мқҙм§„л¶„лҘҳ к°Җм„Өн•ЁмҲҳл“Өм—җ мғҲлЎңмҡҙ лҚ°мқҙн„° xлҘј нҸүк°Җн•ҙліҙлқјн•ң л’Ө, y=1мқј к·ё нҷ•лҘ мқҙ к°ҖмһҘ нҒ° нҒҙлһҳмҠӨлҘј мӮ¬мҡ©н•ҳлҠ” кІғмһ…лӢҲлӢӨ.

мқҙкІғмңјлЎң Classificationм—җ лҢҖн•ң мҶҢк°ңлҘј л§Ҳм№©лӢҲлӢӨ. лӢӨмқҢ нҸ¬мҠӨнҠём—җм„ңлҠ” Regularizationм—җ лҢҖн•ҙ м•Ңм•„ліҙкІ мҠөлӢҲлӢӨ.
