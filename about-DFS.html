DFS의 목적과 시간복잡도 - Purpose of DFS and its Time Complexity

DFS의 목적과 시간복잡도

그래프의 탐색

깊이 우선 탐색(Depth First Search)

사전적 정의

DFS : 그래프의 탐색 방법 중 하나로, 현재 정점과 인접한 간선들 중 아직 방문하지 않은 정점으로 향하는 간선을 무조건 따라간다. 더이상 방문할 곳이 없는 정점에 도달하면, 마지막으로 따라왔던 간선을 따라 뒤로 돌아간다.

가능한 한 그래프에서 “더 깊이” 검색하는 알고리즘으로, 가장 최근에 발견되고 아직 조사하지 않은 간선을 가진 정점 v로부터 나오는 간선을 조사한다. v의 모든 간선이 조사되면 그 검색은 v를 발견하게 해준 정점으로부터 나오는 간선을 조사하기 위해 “뒤로 되돌아간다”. 발견되지 않은 정점이 하나라도 남아있으면, 깊이 우선 탐색은 그 중 하나를 새 출발점으로 선택하고 해당 출발점으로부터의 검색을 반복한다. 이 전체 과정을 모든 정점을 발견할 때까지 반복한다.

  • DFS를 한번만 사용하여 모든 정점을 발견하지 못할 경우가 존재한다. 그래프가 두 개 이상의 컴포넌트로 이루어져있을 경우이다. 따라서, 모든 정점들을 발견하기 위해서는 모든 정점들을 시작점으로 DFS를 해주어야 한다.

    • 이를 저격한 문제가 이분 그래프 문제이다. 그래프가 하나의 컴포넌트로 이루어졌다고 착각하기 쉬운 문제이다.

깊이 우선 탐색의 목적과 쓰임

  • 모든 정점을 (순서대로) 발견

  • 그래프 전체의 구조 파악.

  • 다른 알고리즘의 서브 루틴으로 자주 사용됨.

    • 그래프에서 두 정점이 연결되어있는지를 확인하기

    • 그래프의 연결 요소(Component)의 개수를 세기

    • 위상 정렬(Topological Sort)

    • 오일러 서킷/트레일 찾기(Eulerian Circuit & Trail)

    • 사이클(Cycle)이 존재하는지 확인하기

    • 절단점(Cut Vertex / Articulation Point) 찾기

    • 다리(Bridge) 찾기

    • 강결합 컴포넌트(Strongly Connected Components, SCC)

    • 2 - SAT

DFS의 시간복잡도

  • 인접 리스트(Adjacency List) : O(V + E)

    • 인접 리스트의 경우, 모든 정점에 대해 DFS를 실행하므로 V, 각 간선을 모두 지나므로 E의 시간복잡도를 갖는다.

    • 구체적으로는, 방향있는 그래프의 경우 모든 간선을 한 번씩 확인하여 E, 무방향 그래프의 경우 모든 간선을 두 번씩 확인하여 2E를 갖는다.

    • 이를 빅 O 표기법을 사용하면 모두 O(V + E)를 갖게된다.

  • 인접 행렬(Adjacency Matrix) : O(V * V)

    • 각 정점 V에 대해, 다른 정점들과 연결되어있는 간선이 있는지 모두 탐색하기 때문에 V를 갖는다.

    • 따라서 이 경우 시간복잡도는 O(V * V)이다.

DFS의 구현방법

  • DFS를 구현하는 방법으로는 스택을 이용한 방법이 있다.

  • 방문하는 정점 v를 스택에 넣고, v와 인접한 방문하지 않은 한 정점을 스택에 넣는것을 반복한다.

  • 더 이상 방문할 수 있는 정점이 존재하지 않은 경우에는, 스택에서 빼준다.

  • 스택을 만들어 사용하는 방법과, 함수 호출 스택(재귀)를 이용하는 방법이 존재한다.

스택 구현 vs 재귀 함수 이용

  • 개인적으로 재귀함수를 이용하여 DFS를 구현하는 것을 선호한다. 코딩이 간편하기도 하고, 코드가 직관적이다.

  • 다만, 재귀함수를 이용할 경우에 함수를 호출하고 메모리에 올리는데 드는 오버헤드가 존재하는 단점이 있다.

  • 구체적으로는, 컨텍스트 스위칭(Context Switching)이 일어나 수행시간이 느려진다.

DFS의 구현(코드)

  • Pseudo Code
DFS_All()
{
    for (vertex u in G.V) {
        u.color = WHITE
    }
    time = 0;
    for (vertex u in G.V) {
        if (u.color == WHITE)
            DFS(G, u)
    }
}

DFS(G, u)
{
    time = time + 1 u.d = time;
    u.color = GRAY // mark as visited
        for (vertex u = v in G.Adj[u])
    {
        if (v.color == WHITE)
            DFS(G, v)
    }
    u.color = BALCK // mark as finished
        time = time + 1;
    u.f = time;
}
  • C++ Code
vector<vector<int>> adj;
vector<bool> visited;

void DFS_All()
{
    visited = vector<bool>(adj.size(), false);
    for (int i = 0; i < adj.size(); ++i)
        if (!visited[i])
            DFS(i);
}

void DFS(int node)
{
    cout << "DFS visits " << node << endl;
    visited[node] = true;

    for (int i = 0; i < adj[node].size(); ++i) {
        int next = adj[node][i];

        if (!visited[next])
            DFS(next);
    }
}
